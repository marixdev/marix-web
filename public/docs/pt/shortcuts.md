# Atalhos de Teclado

Lista completa de atalhos de teclado no Marix.

## Notação

| Símbolo | Tecla |
|---------|-------|
| `Ctrl` | Control (Windows/Linux) |
| `⌘` | Command (macOS) |
| `Shift` | Shift |
| `Alt` | Alt/Option |

> **Nota:** No macOS, substitua `Ctrl` por `⌘` (Command).

## Atalhos Globais

| Atalho | Ação |
|--------|------|
| `Ctrl+K` | Abrir Conexão Rápida |
| `Ctrl+Shift+A` | Adicionar Novo Host |
| `Ctrl+Shift+T` | Abrir Terminal Local |
| `Ctrl+Shift+L` | Alternar Descoberta LAN |
| `Ctrl+Shift+O` | Alternar visualização Terminal/SFTP |

## Atalhos do Terminal

| Atalho | Ação |
|--------|------|
| `Ctrl+C` | Copiar (quando texto selecionado) / Cancelar comando |
| `Ctrl+V` | Colar |
| `Ctrl+Shift+C` | Copiar (alternativo) |
| `Ctrl+Shift+V` | Colar (alternativo) |
| `Ctrl+Shift+F` | Buscar no terminal |
| `Ctrl+L` | Limpar tela |
| `Ctrl++` | Aumentar zoom |
| `Ctrl+-` | Diminuir zoom |
| `Ctrl+0` | Resetar zoom |

## Atalhos SFTP

| Atalho | Ação |
|--------|------|
| `Ctrl+Shift+O` | Mudar para SFTP |
| `F2` | Renomear arquivo selecionado |
| `Delete` | Excluir arquivo selecionado |
| `Ctrl+C` | Copiar caminho do arquivo |
| `Ctrl+R` | Atualizar diretório |
| `Backspace` | Ir para diretório pai |

## Atalhos do Editor de Arquivos

| Atalho | Ação |
|--------|------|
| `Ctrl+S` | Salvar arquivo |
| `Ctrl+Z` | Desfazer |
| `Ctrl+Shift+Z` | Refazer |
| `Ctrl+F` | Buscar |
| `Ctrl+H` | Buscar e substituir |
| `Ctrl+G` | Ir para linha |

## Atalhos de Janela

| Atalho | Ação |
|--------|------|
| `F11` | Alternar tela cheia |
| `Ctrl+Shift+D` | Alternar modo escuro/claro |
| `Ctrl+Q` | Sair do aplicativo |

## Personalização

Atualmente, os atalhos de teclado não podem ser personalizados. Este recurso pode ser adicionado em versões futuras.

## Dicas

- A maioria dos atalhos funciona em todas as visualizações
- Atalhos do terminal podem conflitar com atalhos do shell
- Use `Ctrl+Shift+C/V` no terminal para evitar conflitos com `Ctrl+C` (cancelar)

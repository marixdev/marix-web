# Guia de Início Rápido

Conecte-se ao seu primeiro servidor em menos de 5 minutos.

## Adicione Seu Primeiro Host

1. Clique em **adicionar novo host** na lista de servidores
2. Insira os detalhes do servidor:
   - **Nome**: Qualquer nome amigável (ex: "Meu VPS")
   - **Protocolo**: SSH (padrão)
   - **Host**: Endereço IP ou domínio
   - **Porta**: 22 (padrão para SSH)
   - **Usuário**: Seu nome de usuário do servidor
   - **Autenticação**: Senha ou Chave SSH
3. Clique em **Salvar**


## Conecte ao Seu Servidor

1. Clique no nome do host na lista de servidores
2. Você está conectado!

## Dicas Rápidas

### Atalhos do Terminal

| Atalho | Ação |
|--------|------|
| `Ctrl+Shift+C` | Copiar |
| `Ctrl+Shift+V` | Colar |
| `Ctrl+Shift+F` | Buscar |
| `Ctrl+L` | Limpar tela |
| `Ctrl+Plus` | Aumentar zoom |
| `Ctrl+Minus` | Diminuir zoom |

### Suporte Multi-Abas

- **Nova Aba**: Clique em +

### Copiar-Colar

3. **Copiar texto**:
   - Selecione o texto com o mouse → Copiar
   - Ou use `Ctrl+C`
4. **Colar texto**:
   - Clique com o botão direito → Colar
   - Ou use `Ctrl+V`

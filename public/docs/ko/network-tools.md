# 네트워크 도구

Marix에 내장된 네트워크 진단 유틸리티입니다.

## 사용 가능한 도구

| 도구 | 기능 |
|------|------|
| DNS 조회 | 도메인의 DNS 레코드 조회 |
| WHOIS | 도메인 등록 정보 조회 |
| Ping | 호스트 연결 가능 여부 확인 |
| Traceroute | 목적지까지의 네트워크 경로 추적 |
| 포트 스캐너 | 열린 포트 확인 |

## 네트워크 도구 열기

사이드바에서 **"네트워크 도구"**를 선택하세요.

## DNS 조회

도메인의 DNS 레코드를 조회합니다.

### 사용 방법
1. **DNS 조회** 탭 선택
2. 도메인 이름 입력 (예: `example.com`)
3. **조회** 클릭

### 조회 가능한 레코드 유형
- **A** - IPv4 주소
- **AAAA** - IPv6 주소
- **MX** - 메일 서버
- **NS** - 네임 서버
- **TXT** - 텍스트 레코드
- **CNAME** - 별칭 레코드
- **SOA** - 권한 시작 레코드

## WHOIS

도메인 등록 정보를 조회합니다.

### 사용 방법
1. **WHOIS** 탭 선택
2. 도메인 이름 입력
3. **조회** 클릭

### 표시 정보
- 등록 기관
- 생성일
- 만료일
- 네임 서버
- 등록자 정보 (공개된 경우)

## Ping

호스트의 연결 가능 여부와 응답 시간을 확인합니다.

### 사용 방법
1. **Ping** 탭 선택
2. 호스트명 또는 IP 주소 입력
3. **시작** 클릭

### 결과
- 응답 시간 (ms)
- 패킷 손실률
- 평균/최소/최대 지연 시간

## Traceroute

목적지까지의 네트워크 경로를 추적합니다.

### 사용 방법
1. **Traceroute** 탭 선택
2. 대상 호스트 입력
3. **추적** 클릭

### 표시 정보
- 각 홉의 IP 주소
- 각 홉까지의 지연 시간
- 홉 번호

## 포트 스캐너

호스트의 열린 포트를 확인합니다.

### 사용 방법
1. **포트 스캐너** 탭 선택
2. 대상 호스트 입력
3. 포트 범위 설정 (예: 1-1000 또는 22,80,443)
4. **스캔** 클릭

### 일반적인 포트

| 포트 | 서비스 |
|------|--------|
| 21 | FTP |
| 22 | SSH |
| 23 | Telnet |
| 25 | SMTP |
| 53 | DNS |
| 80 | HTTP |
| 110 | POP3 |
| 143 | IMAP |
| 443 | HTTPS |
| 3306 | MySQL |
| 5432 | PostgreSQL |
| 3389 | RDP |

## 사용 팁

### 연결 문제 진단
1. **Ping**으로 호스트 연결 확인
2. **Traceroute**로 경로 문제 확인
3. **포트 스캐너**로 필요한 포트 열림 확인

### 도메인 문제 진단
1. **DNS 조회**로 레코드 확인
2. **WHOIS**로 도메인 상태 확인

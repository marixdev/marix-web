# 빠른 시작 가이드

5분 안에 첫 번째 서버에 연결하세요.

## 첫 번째 호스트 추가

1. 서버 목록에서 **호스트 추가**를 클릭
2. 서버 세부 정보 입력:
   - **이름**: 원하는 이름 (예: "내 VPS")
   - **프로토콜**: SSH (기본값)
   - **호스트**: IP 주소 또는 도메인
   - **포트**: 22 (SSH 기본값)
   - **사용자명**: 서버 사용자명
   - **인증**: 비밀번호 또는 SSH 키
3. **저장** 클릭


## 서버에 연결

1. 서버 목록에서 호스트 이름 클릭
2. 연결 완료!

## 빠른 팁

### 터미널 단축키

| 단축키 | 동작 |
|----------|--------|
| `Ctrl+Shift+C` | 복사 |
| `Ctrl+Shift+V` | 붙여넣기 |
| `Ctrl+Shift+F` | 찾기 |
| `Ctrl+L` | 화면 지우기 |
| `Ctrl+Plus` | 확대 |
| `Ctrl+Minus` | 축소 |

### 멀티 탭 지원

- **새 탭**: + 클릭

### 복사-붙여넣기

3. **텍스트 복사**: 
   - 마우스로 텍스트 선택 → 복사
   - 또는 `Ctrl+C` 사용
4. **텍스트 붙여넣기**: 
   - 마우스 오른쪽 클릭 → 붙여넣기
   - 또는 `Ctrl+V` 사용

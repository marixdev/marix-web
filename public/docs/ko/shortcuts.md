# 키보드 단축키

Marix의 전체 키보드 단축키 목록입니다.

## 표기법

| 기호 | 키 |
|--------|-----|
| `Ctrl` | Control (Windows/Linux) |
| `⌘` | Command (macOS) |
| `Shift` | Shift |
| `Alt` | Alt/Option |

> **참고:** macOS에서는 `Ctrl`을 `⌘` (Command)로 대체하세요.

## 전역 단축키

| 단축키 | 동작 |
|----------|--------|
| `Ctrl+K` | 빠른 연결 열기 |
| `Ctrl+Shift+A` | 새 호스트 추가 |
| `Ctrl+Shift+T` | 로컬 터미널 열기 |
| `Ctrl+Shift+L` | LAN 검색 토글 |
| `Ctrl+Shift+O` | 터미널/SFTP 보기 토글 |

## 터미널 단축키

| 단축키 | 동작 |
|----------|--------|
| `Ctrl+C` | 복사 (텍스트 선택 시) / 명령 취소 |
| `Ctrl+V` | 붙여넣기 |
| `Ctrl+Shift+C` | 복사 (대체) |
| `Ctrl+Shift+V` | 붙여넣기 (대체) |
| `Ctrl+Shift+F` | 터미널에서 찾기 |
| `Ctrl+L` | 화면 지우기 |
| `Ctrl++` | 확대 |
| `Ctrl+-` | 축소 |
| `Ctrl+0` | 확대/축소 재설정 |

## SFTP 단축키

| 단축키 | 동작 |
|----------|--------|
| `Ctrl+Shift+O` | SFTP로 전환 |
| `F2` | 선택한 파일 이름 바꾸기 |
| `Delete` | 선택한 파일 삭제 |
| `Ctrl+C` | 파일 경로 복사 |
| `Ctrl+R` | 디렉토리 새로고침 |
| `Backspace` | 상위 디렉토리로 이동 |

## 파일 편집기 단축키

| 단축키 | 동작 |
|----------|--------|
| `Ctrl+S` | 파일 저장 |
| `Ctrl+Z` | 실행 취소 |
| `Ctrl+Shift+Z` | 다시 실행 |
| `Ctrl+F` | 찾기 |
| `Ctrl+H` | 찾기 및 바꾸기 |
| `Ctrl+G` | 줄로 이동 |

## 창 단축키

| 단축키 | 동작 |
|----------|--------|
| `F11` | 전체 화면 토글 |
| `Ctrl+Shift+D` | 다크/라이트 모드 토글 |
| `Ctrl+Q` | 애플리케이션 종료 |

## 사용자 정의

현재 키보드 단축키는 사용자 정의할 수 없습니다. 이 기능은 향후 버전에서 추가될 수 있습니다.

## 팁

- 대부분의 단축키는 모든 보기에서 작동합니다
- 터미널 단축키가 셸 단축키와 충돌할 수 있습니다
- 터미널에서 `Ctrl+C` (취소)와의 충돌을 피하려면 `Ctrl+Shift+C/V`를 사용하세요

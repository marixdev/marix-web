# 2단계 인증 (2FA/TOTP)

Google Authenticator와 같은 내장 TOTP 코드 생성기. 2FA 비밀키를 저장하고 휴대폰 없이도 코드를 받을 수 있습니다.

## 2FA 인터페이스

![2FA Interface](/2fa.png)

## 2FA 열기

사이드바에서 **"2FA"**를 클릭하세요.

## 새 2FA 계정 추가

### 1단계: 대화 상자 열기
**"계정 추가"** 버튼 클릭

### 2단계: 세부 정보 입력

| 필드 | 설명 | 필수 |
|-------|-------------|----------|
| 계정 이름 | 이 계정의 레이블 (예: "GitHub", "AWS") | 아니오 (자동 이름 지정) |
| 비밀키 | TOTP 비밀키 (QR 코드 설정 페이지에서) | 예 |

### 3단계: 저장
**"추가"** 클릭

## 비밀키 얻는 방법

웹사이트에서 2FA를 활성화할 때:

1. **"QR 코드를 스캔할 수 없나요?"** 또는 **"수동 입력"** 찾기
2. 비밀키 복사 (보통 대문자와 숫자)
3. Marix에 붙여넣기

비밀키 형식 예시:
```
JBSWY3DPEHPK3PXP
```

## 2FA 코드 사용

1. 목록에서 계정 찾기
2. 옆에 6자리 코드가 표시됨
3. 카운트다운 타이머가 코드 새로고침까지 남은 시간 표시
4. **복사** 버튼을 클릭하여 클립보드에 코드 복사

### 타이머
- 코드는 **30초**마다 새로고침
- 원형 진행 표시기가 남은 시간 표시
- 현재 코드가 곧 만료되면 새 코드 대기

## 계정 이름 수정

1. 계정 옆의 **편집** 버튼 클릭
2. 새 이름 입력
3. Enter 누르거나 저장 클릭

## 2FA 계정 삭제

1. **삭제** 버튼 (휴지통 아이콘) 클릭
2. 삭제 확인

> ⚠️ **경고:** 삭제하면 비밀키가 제거됩니다. 삭제하기 전에 연결된 계정에 대한 백업 접근 권한이 있는지 확인하세요.

## 일반적인 서비스

TOTP를 지원하는 인기 서비스:

| 서비스 | 2FA 설정 위치 |
|---------|----------------------|
| GitHub | 설정 → 비밀번호 및 인증 |
| AWS | IAM → 보안 자격 증명 |
| Google | Google 계정 → 보안 |
| Microsoft | 보안 설정 → 2단계 인증 |
| Cloudflare | 프로필 → 인증 |
| DigitalOcean | 계정 → 보안 |
| Linode | 내 프로필 → 비밀번호 및 인증 |

## 모범 사례

1. **비밀키 백업** - 비밀키를 안전하게 저장
2. **복구 코드 보관** - 서비스에서 제공하는 백업 코드 저장
3. **설명적인 이름 사용** - 계정을 빠르게 식별
4. **코드 의존 전 테스트** - 추가 후 즉시 작동 확인
5. **시간 동기화** - 컴퓨터 시계가 정확한지 확인

## 문제 해결

### 코드가 작동하지 않음

1. **시간 동기화 확인** - 컴퓨터 시계가 정확해야 함
2. **비밀키 확인** - 잘못 복사된 경우 다시 입력
3. **다음 코드 대기** - 현재 코드가 만료될 수 있음

### 접근 권한 분실

Marix에 대한 접근 권한을 잃은 경우:
1. 서비스에서 제공한 **복구 코드** 사용
2. 계정 복구를 위해 서비스 지원팀에 문의

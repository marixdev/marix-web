# SSH 터미널

Marix의 SSH 터미널은 원격 서버에 연결하기 위한 강력하고 기능이 풍부한 터미널 에뮬레이터를 제공합니다.

## 터미널 인터페이스

![Terminal Interface](/terminal.png)

SSH로 연결한 후 다음을 볼 수 있습니다:
- 완전한 터미널 에뮬레이터 (xterm.js 기반)
- 상단에 터미널/SFTP 보기 전환 버튼
- 테마 선택 드롭다운
- 재연결 버튼

## 터미널 컨트롤

| 컨트롤 | 기능 |
|---------|----------|
| 터미널 버튼 | 터미널 보기로 전환 |
| SFTP 버튼 | 파일 브라우저로 전환 |
| 테마 선택기 | 터미널 색상 테마 변경 |
| ↺ 재연결 | 연결이 끊어진 경우 재연결 |

## 터미널 사용

1. **명령어 입력**은 일반 SSH 터미널과 동일
2. 명령어는 원격 서버에서 실행됨
3. **텍스트 복사**: 
   - 마우스로 텍스트 선택 → 복사
   - 또는 `Ctrl+C` 사용
4. **텍스트 붙여넣기**: 
   - 마우스 오른쪽 클릭 → 붙여넣기
   - 또는 `Ctrl+V` 사용

## 키보드 단축키

| 단축키 | 동작 |
|----------|--------|
| `Ctrl+C` | 선택된 텍스트 복사 |
| `Ctrl+V` | 클립보드에서 붙여넣기 |
| `Ctrl+L` | 화면 지우기 |
| `Tab` | 명령어 기록 열기/닫기 |

## 터미널 테마 변경

1. 터미널 세션 중
2. 테마 선택기 드롭다운 클릭 (터미널 오른쪽 상단)
3. 사용 가능한 테마 탐색 (400개 이상)
4. 테마를 클릭하여 즉시 적용

사용 가능한 테마 카테고리:
- 클래식 (Tango, Solarized, Monokai)
- 모던 (Dracula, One Dark, Nord)
- 고대비
- 사용자 정의 색상


## 로컬 터미널 열기

원격 서버에 연결하지 않고 로컬 컴퓨터에서 터미널 열기:

- 키보드 단축키: `Ctrl+Shift+T`
- 로컬 명령어 실행에 유용


## 연결 상태

탭에 연결 상태가 표시됩니다:
- **녹색 표시등**: 연결됨
- **빨간색 표시등**: 연결 끊김
- **노란색 표시등**: 연결 중

## 재연결

연결이 끊어진 경우:
Marix가 자동으로 SSH 재연결

재연결 후에도 터미널 스크롤 기록이 유지됩니다.

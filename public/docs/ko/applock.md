# 앱 잠금

앱 잠금으로 Marix 애플리케이션을 보호하세요. 활성화하면 앱이 일정 시간 비활성 후 자동으로 잠겨 서버와 자격 증명에 대한 무단 접근을 방지합니다.

## 앱 잠금 인터페이스

![App Lock Interface](/applock.png)

## 앱 잠금 활성화

1. **설정** (사이드바의 톱니바퀴 아이콘) 열기
2. **보안** 섹션 찾기
3. **앱 잠금 활성화** 토글

## 잠금 방법

앱 잠금 방법 선택:

| 방법 | 설명 | 보안 수준 |
|--------|-------------|----------------|
| **흐림** | 화면 콘텐츠를 흐리게 처리 | 낮음 - 시각적 개인정보만 |
| **PIN** | 잠금 해제에 4-6자리 PIN 필요 | 중간 |
| **비밀번호** | 잠금 해제에 비밀번호 필요 | 높음 |

### 흐림 모드

- 화면 콘텐츠가 흐려짐
- 아무 곳이나 클릭하여 잠금 해제
- 사무실 환경에서 빠른 개인정보 보호에 적합

### PIN 모드

- 4-6자리 숫자 PIN 설정
- 잠금 해제를 위해 PIN 입력
- 보안과 편의성의 좋은 균형

### 비밀번호 모드

- 사용자 정의 비밀번호 설정
- 잠금 해제를 위해 비밀번호 입력
- 가장 높은 보안 수준

## PIN 또는 비밀번호 설정

PIN 또는 비밀번호 모드로 전환 시:

1. 잠금 방법 선택
2. 자격 증명 설정 대화 상자가 나타남
3. PIN/비밀번호 입력
4. 다시 입력하여 확인
5. **저장** 클릭

> **참고:** PIN/비밀번호를 잊어버리면 앱을 재시작해야 합니다. 서버 데이터는 안전하게 유지됩니다.

## 잠금 시간 초과

앱이 잠기기 전까지 대기하는 시간 설정:

| 시간 초과 | 사용 사례 |
|---------|----------|
| 1분 | 최대 보안 |
| 2분 | 높은 보안 |
| 5분 | 기본값, 균형 |
| 10분 | 중간 보안 |
| 15분 | 일상 사용 |
| 30분 | 낮은 보안 |
| 60분 | 최소 보호 |

마우스 이동, 키보드 입력 또는 클릭 시 타이머가 재설정됩니다.

## 잠금 해제

### 흐림 모드
- 화면 아무 곳이나 클릭
- 마우스 이동

### PIN/비밀번호 모드
1. PIN 또는 비밀번호 입력
2. **Enter** 누르기 또는 **잠금 해제** 클릭
3. 앱 즉시 잠금 해제

## 자격 증명 변경

PIN 또는 비밀번호 변경:

1. **설정** → **보안** 이동
2. **PIN 변경** 또는 **비밀번호 변경** 클릭
3. 새 자격 증명 입력
4. 확인 및 저장

## 자격 증명 지우기

PIN/비밀번호를 제거하고 흐림으로 전환:

1. **설정** → **보안** 이동
2. **자격 증명 지우기** 클릭
3. 잠금 방법이 흐림으로 복원됨

## 작동 방식

1. **활동 추적**: Marix가 마우스, 키보드, 클릭 이벤트 모니터링
2. **비활성 타이머**: 설정된 시간 초과 후 활동이 없으면 앱 잠금
3. **즉시 잠금**: 수동으로도 잠금 가능 (구현된 경우)
4. **보안 저장**: PIN/비밀번호는 시스템 키체인을 사용하여 안전하게 저장

## 보안 고려 사항

### 앱 잠금이 보호하는 것

- ✅ 서버 목록에 대한 시각적 접근
- ✅ 저장된 자격 증명 보기
- ✅ 활성 터미널 세션 접근
- ✅ SFTP 파일 관리자 탐색

### 앱 잠금이 보호하지 않는 것

- ❌ 이미 설정된 SSH 연결 (활성 상태 유지)
- ❌ 디스크의 데이터 (이를 위해 시스템 암호화 사용)
- ❌ 네트워크 트래픽 (이를 위해 SSH/TLS 사용)

## 모범 사례

1. 민감한 환경에서는 **비밀번호 모드 사용**
2. 공유 작업 공간에서는 **짧은 시간 초과 설정**
3. **고유한 자격 증명 사용** - 비밀번호 재사용 금지
4. 컴퓨터를 떠나기 전에 **잠금**
5. 최대 보안을 위해 **OS 잠금과 결합**

## 문제 해결

### PIN/비밀번호 분실

- 애플리케이션 재시작
- 앱 잠금이 재시작 시 초기화됨
- 서버 데이터는 보존됨

### 앱이 너무 빨리 잠김

- 설정에서 시간 초과 늘리기
- 활동 감지를 방해할 수 있는 활성 세션 확인

### 앱이 잠기지 않음

- 앱 잠금이 활성화되어 있는지 확인
- 시간 초과 설정 확인
- 앱 창에 포커스가 있는지 확인

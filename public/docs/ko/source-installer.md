# 소스 인스톨러

서버에 인기 있는 프레임워크와 CMS를 설치합니다.

## 지원 소프트웨어

| 카테고리 | 소프트웨어 |
|----------|-----------|
| PHP 프레임워크 | Laravel, Symfony, CodeIgniter |
| CMS | WordPress, Drupal, Joomla |
| Node.js | Express, Next.js, Nuxt.js |
| Python | Django, Flask |
| 정적 사이트 | Hugo, Jekyll |
| 전자상거래 | Magento, PrestaShop, OpenCart |
| 포럼/위키 | phpBB, MediaWiki |

## 소스 인스톨러 열기

1. SFTP로 서버에 연결
2. 원하는 설치 위치로 이동
3. **"소스 설치"** 또는 **"+"** 버튼 클릭
4. 소스 인스톨러 메뉴 선택

## 소프트웨어 설치

### 1단계: 소프트웨어 선택
목록에서 설치할 소프트웨어 선택

### 2단계: 버전 선택
원하는 버전 선택 (최신 버전 권장)

### 3단계: 설치 옵션 설정

| 옵션 | 설명 |
|------|------|
| 설치 경로 | 소프트웨어가 설치될 디렉토리 |
| 데이터베이스 | 필요시 데이터베이스 연결 정보 |
| 관리자 계정 | 초기 관리자 사용자 이름/비밀번호 |

### 4단계: 설치 시작
**"설치"** 버튼 클릭

### 5단계: 완료 대기
- 다운로드 진행률 표시
- 파일 추출 및 설정
- 설치 완료 알림

## 설치 후

소프트웨어에 따라 추가 설정이 필요할 수 있습니다:

### WordPress
1. 웹 브라우저에서 사이트 접속
2. 설치 마법사 완료
3. 관리자 로그인

### Laravel
1. `.env` 파일 설정
2. 데이터베이스 마이그레이션 실행
3. `composer install` (필요시)

### Node.js 앱
1. `npm install` 실행
2. 환경 변수 설정
3. `npm start` 또는 PM2로 실행

## 요구 사항

각 소프트웨어별 서버 요구 사항:

### PHP 기반
- PHP 7.4+ 또는 8.0+
- 필요한 PHP 확장
- Composer (일부 프레임워크)

### Node.js 기반
- Node.js 16+
- npm 또는 yarn

### Python 기반
- Python 3.8+
- pip
- virtualenv 권장

## 문제 해결

### 설치 실패
- 서버 저장 공간 확인
- 쓰기 권한 확인
- PHP/Node 버전 확인

### 권한 오류
- 디렉토리 권한 755 설정
- 파일 권한 644 설정
- 소유자 확인

### 데이터베이스 연결 오류
- 데이터베이스 자격 증명 확인
- 데이터베이스가 존재하는지 확인
- 호스트/포트 확인

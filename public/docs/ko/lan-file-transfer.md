# LAN 파일 전송

Wi-Fi 또는 LAN을 통해 근처 기기 간에 파일을 공유합니다.

## 페어링 코드로 전송

페어링 코드를 사용하여 두 Marix 기기 간에 파일을 전송합니다.

### 보내는 쪽

1. 사이드바에서 **"LAN 파일 전송"** 선택
2. **"파일 보내기"** 탭 선택
3. 보낼 파일 선택
4. 표시된 **페어링 코드** 확인

### 받는 쪽

1. 사이드바에서 **"LAN 파일 전송"** 선택
2. **"파일 받기"** 탭 선택
3. 보내는 쪽의 **페어링 코드** 입력
4. **"연결"** 클릭하여 전송 시작

## 자동 검색

같은 네트워크의 기기가 자동으로 표시됩니다:

1. 기기 목록이 나타남
2. 전송할 기기 선택
3. 수락 요청이 상대 기기에 표시됨

## 파일 유형

전송 가능한 항목:
- 문서 (PDF, Word, Excel 등)
- 이미지
- 동영상
- 압축 파일
- 모든 종류의 파일

## 네트워크 요구 사항

| 요구 사항 | 설명 |
|-------------|-------------|
| 같은 네트워크 | 두 기기가 같은 Wi-Fi/LAN에 있어야 함 |
| 포트 | 내부 포트 자동 사용 |
| 방화벽 | 로컬 네트워크 트래픽 허용 필요 |

## 보안

- **암호화**: 전송 데이터 암호화
- **페어링 코드**: 일회용 코드로 연결
- **승인 필요**: 받는 쪽이 전송을 수락해야 함

## 문제 해결

### 기기가 보이지 않음
- 같은 네트워크에 있는지 확인
- 방화벽 설정 확인
- 앱 재시작

### 전송 실패
- 네트워크 연결 상태 확인
- 저장 공간 확인
- 두 기기에서 앱 재시작

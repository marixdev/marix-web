# SFTP 파일 관리자

SFTP (SSH 파일 전송 프로토콜)는 암호화된 SSH 연결을 통해 안전한 파일 전송을 제공합니다.

## SFTP 열기

### 기존 SSH 연결에서

1. SSH 서버에 연결
2. 세션 상단의 **SFTP** 버튼 클릭
3. 또는 키보드 단축키 사용: `Ctrl+Shift+O`

### 직접 SFTP 연결

1. 호스트 목록에서 SSH 서버를 마우스 오른쪽 클릭
2. 컨텍스트 메뉴에서 **"SFTP"** 선택

## 파일 작업

### 파일 다운로드 (원격 → 로컬)

1. **원격** (오른쪽) 패널에서 파일로 이동

**방법 A: 드래그 앤 드롭**
- 파일을 클릭하여 로컬 (왼쪽) 패널로 드래그
- 원하는 폴더에 놓기

**방법 B: 마우스 오른쪽 클릭**
- 파일을 마우스 오른쪽 클릭
- **"다운로드"** 선택
- 파일이 현재 로컬 폴더에 다운로드

### 파일 업로드 (로컬 → 원격)

1. **로컬** (왼쪽) 패널에서 파일로 이동

**방법 A: 드래그 앤 드롭**
- 파일을 클릭하여 원격 (오른쪽) 패널로 드래그
- 원하는 폴더에 놓기

**방법 B: 마우스 오른쪽 클릭**
- 파일을 마우스 오른쪽 클릭
- **"업로드"** 선택
- 파일이 현재 원격 폴더에 업로드

### 새 폴더 만들기

1. 대상 디렉토리로 이동
2. 빈 공간을 마우스 오른쪽 클릭
3. **"새 폴더"** 선택
4. 폴더 이름 입력
5. 확인 클릭

### 새 파일 만들기

1. 대상 디렉토리로 이동
2. 빈 공간을 마우스 오른쪽 클릭
3. **"새 파일"** 선택
4. 파일 이름 입력
5. 확인 클릭 (빈 파일 생성)

### 파일 또는 폴더 삭제

1. 파일 또는 폴더를 마우스 오른쪽 클릭
2. **"삭제"** 선택
3. 삭제 확인

### 파일 또는 폴더 이름 변경

1. 파일 또는 폴더를 마우스 오른쪽 클릭
2. **"이름 변경"** 선택
3. 새 이름 입력
4. 확인 클릭

## 권한 변경 (chmod)

1. 파일 또는 폴더를 마우스 오른쪽 클릭 (원격 패널)
2. **"권한 변경"** 선택
3. 8진수 형식으로 권한 입력 (예: `755`, `644`)
4. 확인 클릭

### 일반적인 권한 값

| 값 | 의미 | 사용 사례 |
|-------|---------|----------|
| `755` | 소유자: 전체, 기타: 읽기+실행 | 폴더, 실행 파일 |
| `644` | 소유자: 읽기+쓰기, 기타: 읽기 | 일반 파일 |
| `600` | 소유자만 | 개인 파일, SSH 키 |
| `777` | 모든 사용자: 전체 접근 | 권장하지 않음 |

## 원격 파일 편집

1. 원격 패널에서 텍스트 파일 찾기
2. 더블 클릭 (또는 마우스 오른쪽 클릭 → 편집)
3. 내장 편집기에서 파일 열림
4. 변경 사항 적용
5. **저장** 클릭 또는 `Ctrl+S` 누르기
6. 파일이 서버에 저장됨

### 지원되는 파일 유형

**코드 파일:**
`.js`, `.ts`, `.py`, `.php`, `.java`, `.c`, `.cpp`, `.go`, `.rs`, `.rb`

**설정 파일:**
`.json`, `.yml`, `.yaml`, `.toml`, `.ini`, `.conf`, `.env`

**웹 파일:**
`.html`, `.css`, `.scss`, `.vue`, `.svelte`

**스크립트:**
`.sh`, `.bash`, `.ps1`, `.bat`

**기타:**

## 파일 압축

원격 서버에서 파일 또는 폴더로 압축 파일 만들기:

1. 파일 또는 폴더를 마우스 오른쪽 클릭 (원격 패널)
2. **"압축"** 위로 마우스 이동
3. 형식 선택:
   - **ZIP** - 범용, 모든 곳에서 작동
   - **TAR.GZ** - Linux/Unix에 최적
   - **TAR** - 비압축 아카이브

압축 파일은 동일한 디렉토리에 생성됩니다.

## 압축 해제

원격 서버에서 압축 파일 추출:

1. 압축 파일을 마우스 오른쪽 클릭 (원격 패널)
2. **"여기에 압축 해제"** 선택
3. 파일이 현재 디렉토리에 추출됨

### 지원되는 압축 형식

| 형식 | 확장자 |
|--------|------------|
| ZIP | `.zip` |
| TAR | `.tar`, `.tar.gz`, `.tgz`, `.tar.bz2`, `.tar.xz` |
| GZIP | `.gz` |
| BZIP2 | `.bz2` |
| XZ | `.xz` |
| 7-Zip | `.7z` |
| RAR | `.rar` |

> **참고:** 추출에는 서버에 적절한 도구가 설치되어 있어야 합니다 (unzip, tar, 7z, unrar).

## 파일 속성

파일 및 폴더에 대한 자세한 정보 보기:

1. 파일 또는 폴더를 마우스 오른쪽 클릭
2. **"속성"** 선택
3. 세부 정보 보기:

| 속성 | 설명 |
|----------|-------------|
| 이름 | 파일 또는 폴더 이름 |
| 경로 | 전체 경로 위치 |
| 크기 | 바이트/KB/MB/GB 단위 파일 크기 |
| 유형 | 파일 또는 디렉토리 |
| 권한 | 읽기/쓰기/실행 (예: 755) |
| 소유자 | 사용자 소유자 (원격만) |
| 그룹 | 그룹 소유자 (원격만) |
| 수정됨 | 마지막 수정 날짜 |
| 생성됨 | 생성 날짜 (가능한 경우) |
| 접근됨 | 마지막 접근 날짜 |

## 전송 대기열

여러 파일 전송 시:
- 진행률 표시줄에 전체 완료율 표시
- 개별 파일 진행 상황 표시
- ✕를 클릭하여 전송 취소
- 실패한 전송 재시도 가능

## 키보드 단축키

| 단축키 | 동작 |
|----------|--------|
| `Ctrl+Shift+O` | 터미널/SFTP 전환 |
| `F2` | 선택 항목 이름 변경 |
| `Delete` | 선택 항목 삭제 |
| `Ctrl+C` | 경로 복사 |
| `Ctrl+R` | 디렉토리 새로고침 |

## 팁

- **전체 폴더 드래그**로 재귀적 전송
- **더블 클릭**으로 폴더 들어가기
- **검색/필터** 사용으로 빠른 파일 찾기
- **덮어쓰기 확인**으로 실수로 인한 데이터 손실 방지

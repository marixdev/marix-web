# LAN 서버 공유

같은 네트워크의 다른 Marix 기기와 서버 구성을 공유합니다.

## 서버 구성 공유하기

저장된 서버 연결 설정을 다른 기기와 공유합니다.

### 공유하는 쪽

1. 사이드바에서 **"LAN 서버 공유"** 선택
2. 공유할 **서버 선택**
3. 표시된 **페어링 코드** 확인
4. 받는 기기에 코드 전달

### 받는 쪽

1. 사이드바에서 **"LAN 서버 공유"** 선택
2. **"서버 받기"** 탭 선택
3. 공유하는 쪽의 **페어링 코드** 입력
4. **"연결"** 클릭하여 서버 설정 받기

## 공유되는 정보

| 항목 | 공유됨 |
|------|--------|
| 호스트/포트 | ✅ 예 |
| 사용자 이름 | ✅ 예 |
| 인증 방법 | ✅ 예 |
| SSH 키 | ⚠️ 선택적 |
| 비밀번호 | ❌ 아니오 |

> **참고:** 보안을 위해 비밀번호는 공유되지 않습니다.

## 자동 검색

같은 네트워크의 Marix 기기가 자동으로 검색됩니다:

1. 기기 목록에서 대상 선택
2. 공유할 서버 선택
3. 상대 기기에서 수락

## 보안

- 같은 네트워크 내에서만 작동
- 페어링 코드로 인증
- 민감한 정보(비밀번호)는 제외
- 암호화된 전송

## 사용 사례

- 새 기기에 서버 설정 복사
- 팀원과 서버 정보 공유
- 여러 기기 간 동기화

## 문제 해결

### 기기를 찾을 수 없음
- 같은 Wi-Fi/LAN에 연결되어 있는지 확인
- 방화벽이 로컬 트래픽을 차단하지 않는지 확인
- 두 기기에서 Marix 앱 재시작

### 페어링 코드 오류
- 코드가 정확한지 확인
- 코드가 만료되지 않았는지 확인
- 새 코드 생성하여 재시도

# 서버 노트 (스티키 노트)

Marix는 각 서버에 개인 메모를 첨부할 수 있는 **서버 노트** 기능을 제공합니다. 서버 구성 문서화, 유지보수 알림 또는 빠른 참조 정보에 유용합니다.

## 서버 노트 인터페이스

![서버 노트 인터페이스](/note.png)

## 기능

- **서버별 노트**: 각 서버는 자체 노트를 가질 수 있습니다
- **자동 저장**: 입력하는 대로 변경 사항이 자동으로 저장됩니다 (500ms 디바운스)
- **서식 있는 텍스트 지원**: 줄 바꿈이 있는 일반 텍스트
- **시각적 표시기**: 노트가 있는 서버는 채워진 노트 아이콘을 표시합니다
- **서버 데이터와 동기화**: 노트는 서버 백업에 포함됩니다

## 사용법

### 노트 열기

1. 서버에 연결 (SSH 터미널 또는 SFTP)
2. 상단 도구 모음에서 **노트** 버튼 (연필 아이콘) 클릭
3. 오른쪽 하단에 스티키 노트 팝업이 나타납니다

### 노트 작성

- 텍스트 영역에 입력하기만 하면 됩니다
- 노트는 500ms 비활성 후 **자동 저장**됩니다
- "저장 중..." 표시기가 저장 진행 중임을 표시합니다
- `Escape` 누르거나 외부를 클릭하여 닫기

### 시각적 표시기

| 아이콘 상태 | 의미 |
|------------|---------|
| 윤곽선 (비어 있음) | 이 서버에 노트 없음 |
| 채워진 (단색) | 서버에 노트 있음 |
| 호박색/노란색 | 노트 존재 |

## 데이터 저장

- 노트는 서버 구성의 일부로 저장됩니다
- ✅ **로컬 백업에 포함됨** (Marix 백업 기능)
- ✅ **클라우드 백업에 포함됨** (Google Drive, Box)
- ✅ 백업 암호화 사용 시 **암호화됨**

## 키보드 단축키

| 키 | 동작 |
|-----|--------|
| `Escape` | 노트 팝업 닫기 |
| 외부 클릭 | 노트 팝업 닫기 |

## FAQ

### 노트가 암호화되나요?

노트는 서버 구성 파일에 일반 텍스트로 저장됩니다. 암호화된 백업을 생성하면 노트는 모든 서버 데이터와 함께 암호화됩니다.

### 마크다운을 사용할 수 있나요?

노트는 현재 일반 텍스트만 지원합니다. 마크다운 렌더링은 향후 버전에서 추가될 수 있습니다.

### 문자 제한이 있나요?

엄격한 제한은 없지만 매우 긴 노트는 성능에 영향을 줄 수 있습니다. 노트를 간결하게 유지하는 것이 좋습니다.

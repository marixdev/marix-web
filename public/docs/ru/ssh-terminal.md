# SSH-терминал

Полнофункциональный SSH-терминал с поддержкой современных возможностей.

## Обзор

SSH-терминал Marix предоставляет:
- Полноценную эмуляцию терминала
- Поддержку Unicode и эмодзи
- Настраиваемые темы и шрифты
- Множественные вкладки и сессии
- Историю команд
- Поддержку горячих клавиш

## Подключение к серверу

### Быстрое подключение

1. Нажмите **+ Новое подключение**
2. Введите:
   - **Хост** — IP-адрес или доменное имя
   - **Порт** — По умолчанию 22
   - **Имя пользователя** — Логин
   - **Пароль** или **SSH-ключ** — Аутентификация
3. Нажмите **Подключиться**

### Из сохранённых серверов

1. Найдите сервер в списке хостов
2. Нажмите на него или дважды щёлкните
3. Подключение установится автоматически

## Функции терминала

### Копирование и вставка

**Копирование:**
- Выделите текст мышью
- Нажмите `Ctrl+Shift+C` или правый клик → Копировать
- Или текст копируется автоматически при выделении

**Вставка:**
- Нажмите `Ctrl+Shift+V` или правый клик → Вставить
- Или средней кнопкой мыши

### Поиск в терминале

1. Нажмите `Ctrl+Shift+F` или кнопку поиска
2. Введите искомый текст
3. Используйте стрелки для навигации по результатам
4. Нажмите Esc для закрытия поиска

### Масштабирование

- `Ctrl++` — Увеличить шрифт
- `Ctrl+-` — Уменьшить шрифт
- `Ctrl+0` — Сбросить масштаб

### Полноэкранный режим

- Нажмите `F11` для переключения полноэкранного режима
- Или используйте кнопку в панели инструментов

## Настройки терминала

### Внешний вид

| Параметр | Описание |
|----------|----------|
| **Тема** | Цветовая схема терминала |
| **Шрифт** | Моноширинный шрифт |
| **Размер шрифта** | Размер в пикселях |
| **Прозрачность** | Прозрачность фона |
| **Курсор** | Стиль курсора (блок, линия, подчёркивание) |

### Поведение

| Параметр | Описание |
|----------|----------|
| **Буфер прокрутки** | Количество сохраняемых строк |
| **Звуковой сигнал** | Звук при звонке терминала |
| **Мигание курсора** | Анимация курсора |
| **Автокопирование** | Копировать при выделении |

### Кодировка

- По умолчанию UTF-8
- Поддержка других кодировок для устаревших систем

## Работа с сессиями

### Разделение экрана

Разделяйте терминал на несколько панелей:

1. Правый клик → **Разделить по горизонтали**
2. Или правый клик → **Разделить по вертикали**
3. Переключайтесь между панелями кликом или горячими клавишами

### Множественные окна

- Открывайте несколько окон приложения
- Каждое окно независимо
- Синхронизируйте ввод между сессиями (опционально)

### Восстановление сессий

При повторном открытии:
- Восстанавливаются открытые вкладки
- Восстанавливается буфер прокрутки
- Автоподключение к серверам (опционально)

## Специальные функции

### Отправка команд

Отправка команды на несколько серверов одновременно:

1. Откройте несколько вкладок с подключениями
2. Включите **Синхронный ввод**
3. Команды отправляются на все выбранные сессии

### Логирование сессии

Сохранение вывода терминала в файл:

1. Правый клик → **Начать логирование**
2. Укажите файл для сохранения
3. Все данные записываются в файл
4. Правый клик → **Остановить логирование**

### Макросы

Запись и воспроизведение последовательностей команд:

1. **Меню** → **Макросы** → **Записать**
2. Выполните команды
3. **Макросы** → **Остановить запись**
4. Сохраните макрос с именем
5. Воспроизводите одним кликом

## Интеграция с другими функциями

### SFTP

- Нажмите **SFTP** для открытия файлового менеджера
- Быстрый переход к текущей директории
- Загрузка и скачивание файлов

### Сниппеты

- Быстрая вставка сохранённых команд
- Доступ через меню или горячие клавиши

### История команд

- Просмотр истории выполненных команд
- Поиск по истории
- Повторное выполнение команд

## Устранение неполадок

### Проблемы с отображением

- Проверьте кодировку (UTF-8)
- Убедитесь, что шрифт поддерживает нужные символы
- Попробуйте другую тему

### Задержка ввода

- Проверьте сетевое соединение
- Отключите сжатие, если оно вызывает проблемы
- Попробуйте другой алгоритм шифрования

### Обрыв соединения

- Настройте Keep-Alive интервал
- Проверьте настройки сервера
- Убедитесь в стабильности сети

## Советы

- Используйте горячие клавиши для ускорения работы
- Настройте внешний вид под свои предпочтения
- Сохраняйте часто используемые команды в сниппетах
- Используйте логирование для важных сессий

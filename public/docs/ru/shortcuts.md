# Горячие клавиши

Marix поддерживает различные горячие клавиши для быстрой навигации и работы.

## Глобальные горячие клавиши

| Сочетание | Действие |
|-----------|----------|
| `Ctrl+N` | Новое SSH-подключение |
| `Ctrl+T` | Новая вкладка |
| `Ctrl+W` | Закрыть текущую вкладку |
| `Ctrl+Tab` | Следующая вкладка |
| `Ctrl+Shift+Tab` | Предыдущая вкладка |
| `Ctrl+1-9` | Переключиться на вкладку 1-9 |
| `F11` | Полноэкранный режим |

## Горячие клавиши терминала

| Сочетание | Действие |
|-----------|----------|
| `Ctrl+C` | Прервать текущую команду |
| `Ctrl+Z` | Приостановить процесс |
| `Ctrl+D` | Выход/EOF |
| `Ctrl+L` | Очистить экран |
| `Ctrl+A` | Перейти в начало строки |
| `Ctrl+E` | Перейти в конец строки |
| `Ctrl+U` | Удалить строку до курсора |
| `Ctrl+K` | Удалить строку после курсора |
| `Ctrl+W` | Удалить слово до курсора |
| `Ctrl+R` | Поиск в истории команд |
| `Tab` | Автодополнение |
| `↑` / `↓` | Навигация по истории команд |

## Горячие клавиши SFTP

| Сочетание | Действие |
|-----------|----------|
| `Ctrl+Shift+O` | Переключить Терминал/SFTP |
| `F2` | Переименовать |
| `Delete` | Удалить |
| `Ctrl+C` | Копировать путь |
| `Ctrl+R` | Обновить |
| `Backspace` | К родительской директории |

## Горячие клавиши редактора

| Сочетание | Действие |
|-----------|----------|
| `Ctrl+S` | Сохранить |
| `Ctrl+Z` | Отменить |
| `Ctrl+Y` | Повторить |
| `Ctrl+F` | Поиск |
| `Ctrl+H` | Найти и заменить |
| `Ctrl+G` | Перейти к строке |
| `Ctrl+/` | Переключить комментарий |

## Настройка горячих клавиш

Вы можете настроить горячие клавиши в настройках:

1. Перейдите в **Настройки**
2. Выберите **Горячие клавиши**
3. Найдите нужное действие
4. Нажмите для изменения сочетания
5. Введите новое сочетание клавиш
6. Нажмите **Сохранить**

## Советы

- Используйте `Ctrl+Shift+O` для быстрого переключения между терминалом и SFTP
- `Ctrl+R` в терминале позволяет искать команды в истории
- Многие сочетания можно переназначить под ваши предпочтения

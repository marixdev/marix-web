# Блокировка приложения

Защитите приложение Marix с помощью блокировки. При включении приложение автоматически блокируется после периода бездействия, предотвращая несанкционированный доступ к вашим серверам и учётным данным.

## Интерфейс блокировки приложения

![Интерфейс блокировки приложения](/applock.png)

## Включение блокировки приложения

1. Откройте **Настройки** (значок шестерёнки в боковой панели)
2. Найдите раздел **Безопасность**
3. Включите **Блокировку приложения**

## Методы блокировки

Выберите способ блокировки приложения:

| Метод | Описание | Уровень безопасности |
|-------|----------|----------------------|
| **Размытие** | Размывает содержимое экрана | Низкий — только визуальная конфиденциальность |
| **PIN-код** | Требует 4-6 значный PIN для разблокировки | Средний |
| **Пароль** | Требует пароль для разблокировки | Высокий |

### Режим размытия

- Содержимое экрана становится размытым
- Нажмите в любом месте для разблокировки
- Лучше всего для быстрой конфиденциальности в офисе

### Режим PIN-кода

- Установите 4-6 значный цифровой PIN
- Введите PIN для разблокировки
- Хороший баланс безопасности и удобства

### Режим пароля

- Установите пользовательский пароль
- Введите пароль для разблокировки
- Высший уровень безопасности

## Настройка PIN-кода или пароля

При переключении на режим PIN или пароля:

1. Выберите метод блокировки
2. Появится диалог для установки учётных данных
3. Введите ваш PIN/пароль
4. Подтвердите, введя его повторно
5. Нажмите **Сохранить**

> **Примечание:** Если вы забудете PIN/пароль, вам нужно будет перезапустить приложение. Ваши данные серверов останутся в безопасности.

## Тайм-аут блокировки

Настройте время ожидания перед блокировкой:

| Тайм-аут | Применение |
|----------|------------|
| 1 минута | Максимальная безопасность |
| 2 минуты | Высокая безопасность |
| 5 минут | По умолчанию, сбалансировано |
| 10 минут | Умеренная безопасность |
| 15 минут | Повседневное использование |
| 30 минут | Низкая безопасность |
| 60 минут | Минимальная защита |

Таймер сбрасывается при любом движении мыши, вводе с клавиатуры или нажатии.

## Разблокировка

### Режим размытия
- Нажмите в любом месте экрана
- Переместите мышь

### Режим PIN/пароля
1. Введите ваш PIN или пароль
2. Нажмите **Enter** или кнопку **Разблокировать**
3. Приложение разблокируется мгновенно

## Изменение учётных данных

Чтобы изменить PIN или пароль:

1. Перейдите в **Настройки** → **Безопасность**
2. Нажмите **Изменить PIN** или **Изменить пароль**
3. Введите новые учётные данные
4. Подтвердите и сохраните

## Очистка учётных данных

Чтобы удалить PIN/пароль и вернуться к размытию:

1. Перейдите в **Настройки** → **Безопасность**
2. Нажмите **Очистить учётные данные**
3. Метод блокировки вернётся к размытию

## Как это работает

1. **Отслеживание активности**: Marix отслеживает события мыши, клавиатуры и нажатия
2. **Таймер бездействия**: После отсутствия активности в течение установленного времени приложение блокируется
3. **Мгновенная блокировка**: Вы также можете заблокировать вручную (если реализовано)
4. **Безопасное хранение**: PIN/пароль хранится безопасно с использованием системной связки ключей

## Соображения безопасности

### Что защищает блокировка приложения

- ✅ Визуальный доступ к списку серверов
- ✅ Просмотр сохранённых учётных данных
- ✅ Доступ к активным терминальным сессиям
- ✅ Просмотр SFTP файловых менеджеров

### Что НЕ защищает блокировка приложения

- ❌ Уже установленные SSH-подключения (они остаются активными)
- ❌ Данные на диске (используйте системное шифрование)
- ❌ Сетевой трафик (используйте SSH/TLS)

## Лучшие практики

1. **Используйте режим пароля** для чувствительных сред
2. **Установите короткие тайм-ауты** в общих рабочих пространствах
3. **Используйте уникальные учётные данные** — не используйте пароли повторно
4. **Блокируйте перед уходом** от компьютера
5. **Комбинируйте с блокировкой ОС** для максимальной безопасности

## Устранение неполадок

### Забыли PIN/пароль

- Перезапустите приложение
- Блокировка приложения сбрасывается при перезапуске
- Данные серверов сохраняются

### Приложение блокируется слишком быстро

- Увеличьте тайм-аут в настройках
- Проверьте, есть ли активные сессии, которые могут препятствовать обнаружению активности

### Приложение не блокируется

- Убедитесь, что блокировка приложения включена
- Проверьте настройку тайм-аута
- Убедитесь, что окно приложения в фокусе

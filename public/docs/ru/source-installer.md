# Установщик исходников фреймворков

Быстро устанавливайте популярные фреймворки и CMS прямо на ваш сервер.

## Обзор

Установщик исходников позволяет:
- Устанавливать популярные фреймворки одним кликом
- Автоматически загружать последние версии
- Распаковывать файлы в нужную директорию
- Экономить время на ручной загрузке и настройке

## Использование

1. Подключитесь к серверу через SSH или SFTP
2. Перейдите в **Меню** → **Установщик исходников**
3. Выберите фреймворк из списка
4. Укажите директорию для установки
5. Нажмите **Установить**
6. Дождитесь завершения загрузки и распаковки

## Процесс установки

### Шаг 1: Выбор фреймворка

1. Откройте установщик исходников
2. Просмотрите доступные фреймворки
3. Нажмите на нужный фреймворк

### Шаг 2: Настройка

1. Укажите путь установки (например, `/var/www/html/myproject`)
2. Выберите версию (последняя по умолчанию)
3. Проверьте параметры

### Шаг 3: Установка

1. Нажмите **Установить**
2. Наблюдайте за прогрессом загрузки
3. Дождитесь распаковки файлов
4. Получите уведомление о завершении

## После установки

### Настройка прав доступа

Для большинства фреймворков требуется настройка прав:

```bash
# Laravel
chmod -R 755 storage bootstrap/cache
chown -R www-data:www-data storage bootstrap/cache

# WordPress
chmod -R 755 wp-content
chown -R www-data:www-data wp-content
```

### Настройка базы данных

1. Создайте базу данных
2. Создайте пользователя
3. Настройте подключение в конфигурации фреймворка

### Настройка веб-сервера

Настройте виртуальный хост для вашего фреймворка:

**Apache:**
```apache
<VirtualHost *:80>
    ServerName example.com
    DocumentRoot /var/www/html/myproject/public
    
    <Directory /var/www/html/myproject/public>
        AllowOverride All
        Require all granted
    </Directory>
</VirtualHost>
```

**Nginx:**
```nginx
server {
    listen 80;
    server_name example.com;
    root /var/www/html/myproject/public;
    
    location / {
        try_files $uri $uri/ /index.php?$query_string;
    }
}
```

Перед установкой убедитесь, что на сервере установлены:

| Компонент | Требование |

| pip | Для Python-проектов |

## Устранение неполадок

### Ошибка загрузки

- Проверьте интернет-соединение
- Проверьте наличие свободного места на диске
- Попробуйте загрузку позже

### Ошибка распаковки

- Убедитесь, что `unzip` или `tar` установлены на сервере
- Проверьте права на запись в целевую директорию

### Ошибка прав доступа

- Запустите команды chmod и chown
- Проверьте владельца директории

## Советы

- Всегда делайте резервную копию перед установкой
- Проверяйте совместимость версий фреймворка с вашим сервером
- Читайте официальную документацию фреймворка после установки
- Используйте версии LTS для производственных серверов

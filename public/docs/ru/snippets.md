# Сниппеты команд

Сохраняйте часто используемые команды как сниппеты для быстрого доступа.

## Обзор

Сниппеты команд позволяют:
- Сохранять часто используемые команды
- Организовывать команды по категориям
- Быстро вставлять команды в терминал
- Делиться командами между устройствами

## Создание сниппета

1. Перейдите в **Меню** → **Сниппеты**
2. Нажмите **+ Добавить сниппет**
3. Заполните:
   - **Название** — Описательное имя
   - **Команда** — Сама команда
   - **Категория** — Группировка (опционально)
   - **Описание** — Дополнительные заметки (опционально)
4. Нажмите **Сохранить**

## Использование сниппетов

### Из панели сниппетов

1. Откройте **Меню** → **Сниппеты**
2. Найдите нужный сниппет
3. Нажмите на него для вставки в терминал
4. Или нажмите кнопку копирования для копирования в буфер

### Быстрый доступ

- Используйте поиск для быстрого поиска сниппетов
- Избранные сниппеты появляются вверху списка
- Недавно использованные сниппеты легко доступны

## Управление сниппетами

### Редактирование

1. Найдите сниппет в списке
2. Нажмите кнопку **Редактировать** (карандаш)
3. Измените нужные поля
4. Нажмите **Сохранить**

### Удаление

1. Найдите сниппет в списке
2. Нажмите кнопку **Удалить** (корзина)
3. Подтвердите удаление

### Категории

Организуйте сниппеты по категориям:
- **Системные** — Системные команды
- **Сетевые** — Сетевые утилиты
- **Файловые** — Работа с файлами
- **Git** — Команды Git
- **Docker** — Команды Docker
- **Пользовательские** — Ваши категории

## Примеры полезных сниппетов

### Системная информация
```bash
uname -a && cat /etc/os-release
```

### Использование диска
```bash
df -h | grep -v tmpfs
```

### Использование памяти
```bash
free -h
```

### Активные подключения
```bash
netstat -tuln | grep LISTEN
```

### Поиск больших файлов
```bash
find / -type f -size +100M 2>/dev/null
```

### Docker статус
```bash
docker ps -a && docker images
```

### Git статус
```bash
git status && git log --oneline -5
```

## Переменные в сниппетах

Вы можете использовать переменные в сниппетах:

| Переменная | Описание |
|------------|----------|
| `${input}` | Запрос ввода при выполнении |
| `${date}` | Текущая дата |
| `${time}` | Текущее время |
| `${host}` | Имя текущего хоста |

### Пример с переменной
```bash
grep "${input}" /var/log/syslog
```

При выполнении будет запрошен ввод для подстановки.

## Импорт и экспорт

### Экспорт сниппетов

1. Перейдите в **Сниппеты** → **Настройки**
2. Нажмите **Экспорт**
3. Выберите место сохранения
4. Файл сохраняется в формате JSON

### Импорт сниппетов

1. Перейдите в **Сниппеты** → **Настройки**
2. Нажмите **Импорт**
3. Выберите файл JSON
4. Сниппеты добавляются к существующим

## Синхронизация

При включённой синхронизации через облако:
- Сниппеты синхронизируются между устройствами
- Изменения применяются автоматически
- Резервная копия сохраняется в облаке

## Советы

- Используйте понятные названия для сниппетов
- Группируйте связанные команды по категориям
- Добавляйте описания для сложных команд
- Регулярно экспортируйте для резервного копирования

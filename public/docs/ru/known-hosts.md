# Известные хосты

Известные хосты хранят отпечатки серверов для обнаружения изменений идентификации сервера (потенциальная проблема безопасности).

## Интерфейс известных хостов

![Интерфейс известных хостов](/knowhosts.png)

## Что такое известные хосты?

При первом подключении к SSH-серверу сервер предоставляет свой отпечаток. Marix сохраняет его для:
- Проверки идентификации сервера при будущих подключениях
- Обнаружения атак «человек посередине»
- Оповещения при неожиданном изменении ключа сервера

## Открытие известных хостов

Нажмите **«Известные хосты»** в боковой панели.

## Просмотр известных хостов

Список показывает:

| Столбец | Описание |
|---------|----------|
| Хост | Имя хоста/IP сервера |
| Порт | SSH-порт |
| Тип ключа | Алгоритм SSH-ключа (ED25519, RSA, ECDSA) |
| Отпечаток | Уникальный идентификатор сервера |
| Добавлен | Когда хост был доверен |

## Поиск известных хостов

1. Используйте поле поиска вверху
2. Ищите по имени хоста, IP или отпечатку

## Фильтр по типу ключа

1. Используйте выпадающий фильтр
2. Выберите: Все, ED25519, RSA или ECDSA

## Импорт хоста с сервера

Вручную добавьте отпечаток сервера без подключения:

1. Нажмите кнопку **«Импорт с хоста»**
2. Введите:
   - Имя хоста или IP
   - Порт (по умолчанию 22)
3. Нажмите **«Импорт»**
4. Отпечаток сервера будет получен и сохранён

## Удаление известного хоста

1. Найдите хост в списке
2. Нажмите кнопку **Удалить** (значок корзины)
3. Подтвердите удаление

### Когда удалять

- Сервер был переустановлен и имеет новый ключ
- IP/имя хоста сервера изменились
- Вы больше не подключаетесь к этому серверу

## Проверка ключа хоста

При подключении к новому серверу:
1. Marix получает отпечаток сервера
2. Вас просят проверить и доверить ему
3. После принятия он сохраняется в известных хостах
4. Будущие подключения проверяются автоматически

## Предупреждения безопасности

### «Ключ хоста изменился»
Это предупреждение появляется, когда отпечаток сервера отличается от сохранённого.

**Возможные причины:**
- Сервер был переустановлен
- SSH-ключ сервера был перегенерирован
- ⚠️ **Атака «человек посередине»** (редко, но возможно)

**Что делать:**
1. Уточните у администратора сервера, действительно ли ключ изменился
2. Если подтверждено, удалите старую запись из известных хостов
3. Переподключитесь и примите новый отпечаток
4. Если подозрительно, не подключайтесь

## Лучшие практики

1. **Проверяйте отпечатки** при подключении к новым серверам
2. **Не игнорируйте предупреждения** об изменённых ключах хостов
3. **Регулярно просматривайте** ваш список известных хостов
4. **Удаляйте старые записи** для серверов, которые больше не используете

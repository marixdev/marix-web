# Руководство по быстрому старту

Подключитесь к первому серверу менее чем за 5 минут.

## Добавление первого хоста

1. Нажмите **добавить новый хост** в списке серверов
2. Введите данные сервера:
   - **Имя**: Любое понятное имя (например, «Мой VPS»)
   - **Протокол**: SSH (по умолчанию)
   - **Хост**: IP-адрес или домен
   - **Порт**: 22 (по умолчанию для SSH)
   - **Имя пользователя**: Имя пользователя сервера
   - **Аутентификация**: Пароль или SSH-ключ
3. Нажмите **Сохранить**


## Подключение к серверу

1. Нажмите на имя хоста в списке серверов
2. Вы подключены!

## Быстрые советы

### Горячие клавиши терминала

| Сочетание | Действие |
|-----------|----------|
| `Ctrl+Shift+C` | Копировать |
| `Ctrl+Shift+V` | Вставить |
| `Ctrl+Shift+F` | Поиск |
| `Ctrl+L` | Очистить экран |
| `Ctrl+Plus` | Увеличить |
| `Ctrl+Minus` | Уменьшить |

### Поддержка нескольких вкладок

- **Новая вкладка**: Нажмите +

### Копирование-вставка

3. **Копирование текста**: 
   - Выделите текст мышью → Копировать
   - Или используйте `Ctrl+C`
4. **Вставка текста**: 
   - Правый клик → Вставить
   - Или используйте `Ctrl+V`

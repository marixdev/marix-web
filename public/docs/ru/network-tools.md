# Сетевые инструменты

Полный набор инструментов сетевой диагностики, встроенных в Marix.

![Интерфейс сетевых инструментов](/loockup.png)

## Открытие сетевых инструментов

Нажмите **«Поиск»** в боковой панели.

## Использование сетевого инструмента

### Шаг 1: Выберите инструмент
Нажмите на инструмент в левой боковой панели.

### Шаг 2: Введите цель
- Доменное имя, IP-адрес или имя хоста
- Для TCP-порта: также введите номер порта

### Шаг 3: Запустите
Нажмите кнопку **«Запустить»** или Enter.

### Шаг 4: Просмотрите результаты
- **Зелёный значок** = Успех
- **Красный значок** = Ошибка
- Временная метка показывает время выполнения

## DNS-инструменты

### A-запись
Возвращает IPv4-адрес(а) для домена.

```
example.com → 93.184.216.34
```

### AAAA-запись
Возвращает IPv6-адрес(а) для домена.

```
example.com → 2606:2800:220:1:248:1893:25c8:1946
```

### MX-запрос
Возвращает почтовые серверы для домена.

```
example.com →
  10 mail1.example.com
  20 mail2.example.com
```

### TXT-запись
Возвращает текстовые записи (часто для верификации, SPF, DKIM).

```
example.com → "v=spf1 include:_spf.google.com ~all"
```

### CNAME-запрос
Возвращает каноническое имя (псевдоним) для поддомена.

```
www.example.com → example.com
```

### NS-запрос
Возвращает серверы имён для домена.

```
example.com →
  ns1.example.com
  ns2.example.com
```

### PTR-запрос (Обратный DNS)
Возвращает имя хоста для IP-адреса.

```
8.8.8.8 → dns.google
```

## Сетевая диагностика

### Ping
Проверка подключения и измерение задержки.

```
Пинг example.com:
  Ответ: 64 байта, время=23мс
  Ответ: 64 байта, время=21мс
  Среднее: 22мс
```

### Traceroute
Трассировка сетевого пути до назначения.

```
1  router.local (192.168.1.1) 1мс
2  isp-gateway (10.0.0.1) 5мс
3  backbone.isp.net 12мс
...
```

### Тест TCP-порта
Проверка открытости конкретного порта.

**Быстрые кнопки портов:** SSH(22), HTTP(80), HTTPS(443), MySQL(3306) и т.д.

```
example.com:443 → Открыт
example.com:22 → Закрыт/Фильтруется
```

### Web-проверка
Тест HTTP/HTTPS подключения и ответа.

```
https://example.com
  Статус: 200 OK
  Время ответа: 145мс
  SSL: Действителен
```

## IP-инструменты

### IP Info
Получение информации о владельце и местоположении IP.

```
8.8.8.8 →
  Организация: Google LLC
  Страна: США
  ASN: AS15169
```

### Проверка чёрного списка
Проверка IP на наличие в спам/abuse чёрных списках.

```
Проверка 1.2.3.4:
  ✅ Spamhaus: Чисто
  ✅ Barracuda: Чисто
  ⚠️ SORBS: В списке
```

## Инструменты для доменов

### WHOIS
Получение информации о регистрации домена.

```
example.com →
  Регистратор: Example Registrar
  Создан: 1995-08-14
  Истекает: 2025-08-13
  Статус: clientTransferProhibited
```

### DNS-проверка
Комплексная проверка здоровья DNS.

```
Проверка example.com:
  ✅ NS-записи найдены
  ✅ A-запись разрешается
  ✅ MX-записи настроены
  ✅ SPF-запись действительна
```

## Советы

- **Копируйте результаты** выделением и `Ctrl+C`
- **Выполняйте несколько проверок** для одного домена
- Результаты показывают **временные метки** для справки
- Используйте **IP Info** для исследования неизвестных IP
- **Проверка чёрного списка** перед отправкой писем с нового IP

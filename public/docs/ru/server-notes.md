# Заметки сервера (Стикеры)

Marix предоставляет функцию **Заметки сервера**, которая позволяет прикреплять личные заметки к каждому серверу. Это полезно для документирования конфигураций сервера, напоминаний о техническом обслуживании или быстрой справочной информации.

## Интерфейс заметок сервера

![Интерфейс заметок сервера](/note.png)

## Возможности

- **Заметки для каждого сервера**: Каждый сервер может иметь свои заметки
- **Автосохранение**: Изменения сохраняются автоматически при вводе (задержка 500мс)
- **Поддержка форматированного текста**: Простой текст с переносами строк
- **Визуальный индикатор**: Серверы с заметками показывают заполненный значок заметки
- **Синхронизация с данными сервера**: Заметки включаются в резервные копии сервера

## Использование

### Открытие заметок

1. Подключитесь к серверу (SSH-терминал или SFTP)
2. Нажмите кнопку **Заметки** на верхней панели инструментов (значок карандаша)
3. Всплывающий стикер появится в правом нижнем углу

### Написание заметок

- Просто вводите в текстовую область
- Заметки **автоматически сохраняются** после 500мс бездействия
- Индикатор «Сохранение...» показывает процесс сохранения
- Нажмите `Escape` или кликните снаружи для закрытия

### Визуальные индикаторы

| Состояние значка | Значение |
|------------------|----------|
| Контур (пустой) | Нет заметок для этого сервера |
| Заполненный (сплошной) | Сервер имеет заметки |
| Янтарный/Жёлтый цвет | Заметки существуют |

## Хранение данных

- Заметки хранятся как часть конфигурации сервера
- ✅ **Включаются в локальные резервные копии** (функция резервного копирования Marix)
- ✅ **Включаются в облачные резервные копии** (Google Drive, Box)
- ✅ **Шифруются** при использовании шифрования резервных копий

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `Escape` | Закрыть всплывающее окно заметки |
| Клик снаружи | Закрыть всплывающее окно заметки |

## Часто задаваемые вопросы

### Заметки шифруются?

Заметки хранятся как простой текст в файле конфигурации сервера. При создании зашифрованной резервной копии заметки шифруются вместе со всеми данными сервера.

### Могу ли я использовать Markdown?

Заметки в настоящее время только простой текст. Рендеринг Markdown может быть добавлен в будущей версии.

### Есть ли ограничение на количество символов?

Жёсткого ограничения нет, но очень длинные заметки могут повлиять на производительность. Рекомендуется делать заметки краткими.

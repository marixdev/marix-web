# Mga Keyboard Shortcut

Ang Marix ay sumusuporta sa mga keyboard shortcut para sa mabilis na pag-access sa mga tampok at aksyon.

## Mga Global Shortcut

Mga shortcut na gumagana sa buong application:

| Shortcut | Aksyon |
|----------|--------|
| `Ctrl+N` | Bagong SSH connection |
| `Ctrl+W` | Isara ang kasalukuyang tab |
| `Ctrl+Tab` | Lumipat sa susunod na tab |
| `Ctrl+Shift+Tab` | Lumipat sa nakaraang tab |
| `Ctrl+1` hanggang `Ctrl+9` | Lumipat sa tab na numero 1-9 |
| `Ctrl+,` | Buksan ang Settings |
| `Ctrl+K` | Buksan ang Command Palette |
| `F11` | I-toggle ang Full Screen |

## Mga Terminal Shortcut

Mga shortcut na gumagana sa SSH Terminal:

| Shortcut | Aksyon |
|----------|--------|
| `Ctrl+Shift+C` | Kopyahin ang napiling teksto |
| `Ctrl+Shift+V` | I-paste ang teksto |
| `Ctrl+Shift+F` | Maghanap sa terminal |
| `Ctrl+Shift+O` | Lumipat sa SFTP mode |
| `Ctrl+Shift+N` | Magbukas ng bagong terminal window |
| `Ctrl+L` | I-clear ang terminal screen |
| `Ctrl+D` | Magpadala ng EOF/Magsara ng shell |
| `↑` / `↓` | I-navigate ang command history |

## Mga SFTP Shortcut

Mga shortcut na gumagana sa SFTP File Manager:

| Shortcut | Aksyon |
|----------|--------|
| `F2` | Palitan ng pangalan ang napiling file |
| `Delete` | Burahin ang napiling file o folder |
| `Ctrl+C` | Kopyahin ang file path |
| `Ctrl+R` | I-refresh ang directory |
| `Enter` | Buksan ang folder o file |
| `Backspace` | Pumunta sa parent directory |
| `Ctrl+A` | Piliin ang lahat ng file |

## Mga Editor Shortcut

Mga shortcut na gumagana sa built-in Text Editor:

| Shortcut | Aksyon |
|----------|--------|
| `Ctrl+S` | I-save ang file |
| `Ctrl+Z` | I-undo |
| `Ctrl+Y` | I-redo |
| `Ctrl+F` | Maghanap |
| `Ctrl+H` | Hanapin at palitan |
| `Ctrl+G` | Pumunta sa linya |
| `Ctrl+/` | I-toggle ang comment |
| `Ctrl+D` | Kopyahin ang linya |
| `Ctrl+Shift+K` | Burahin ang linya |

## Mga Database Shortcut

Mga shortcut na gumagana sa Database Client:

| Shortcut | Aksyon |
|----------|--------|
| `Ctrl+Enter` | Patakbuhin ang query |
| `Ctrl+Shift+Enter` | Patakbuhin ang napiling query |
| `F5` | I-refresh ang mga table |
| `Ctrl+Space` | Autocomplete |

## Pag-customize ng mga Shortcut

Kasalukuyang, ang mga keyboard shortcut ay fixed at hindi maaaring i-customize. Ang future na bersyon ay maaaring mag-alok ng customizable na mga keybinding.

## Mga Platform-Specific na Note

- Sa **macOS**, gamitin ang `Cmd` sa halip na `Ctrl`
- Sa **Linux**, ang ilang shortcut ay maaaring mag-conflict sa system keybinding
- Ang mga function key (`F1`-`F12`) ay maaaring kailanganin ang `Fn` key sa ilang keyboard

## Mga Tip

- Gamitin ang `Ctrl+K` para mabilis na maghanap ng mga utos gamit ang Command Palette
- Gamitin ang `Ctrl+Tab` para mabilis na lumipat sa pagitan ng mga bukas na session
- Gamitin ang `Ctrl+Shift+O` para lumipat sa pagitan ng Terminal at SFTP mode

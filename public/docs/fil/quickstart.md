# Gabay sa Mabilisang Pagsisimula

Kumonekta sa iyong unang server sa loob ng 5 minuto.

## Idagdag ang Iyong Unang Host

1. I-click ang **add new host** sa server list 
2. Ilagay ang mga detalye ng server:
   - **Name**: Kahit anong pangalan (hal., "My VPS")
   - **Protocol**: SSH (default)
   - **Host**: IP address o domain
   - **Port**: 22 (default para sa SSH)
   - **Username**: Ang iyong server username
   - **Authentication**: Password o SSH Key
3. I-click ang **Save**


## Kumonekta sa Iyong Server

1. I-click ang pangalan ng host sa server list
2. Nakakonekta ka na!

## Mga Mabilisang Tip

### Mga Terminal Shortcut

| Shortcut | Aksyon |
|----------|--------|
| `Ctrl+Shift+C` | Kopyahin |
| `Ctrl+Shift+V` | I-paste |
| `Ctrl+Shift+F` | Maghanap |
| `Ctrl+L` | I-clear ang screen |
| `Ctrl+Plus` | Mag-zoom in |
| `Ctrl+Minus` | Mag-zoom out |

### Multi-Tab Support

- **Bagong Tab**: I-click ang + 

### Copy-Paste 

3. **Kopyahin ang text**: 
   - Pumili ng text gamit ang mouse → Kopyahin
   - O gamitin ang `Ctrl+C`
4. **I-paste ang text**: 
   - Right-click → Paste
   - O gamitin ang `Ctrl+V`

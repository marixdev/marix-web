# Phím tắt

Danh sách đầy đủ các phím tắt trong Marix.

## Ký hiệu

| Ký hiệu | Phím |
|---------|------|
| `Ctrl` | Control (Windows/Linux) |
| `⌘` | Command (macOS) |
| `Shift` | Shift |
| `Alt` | Alt/Option |

> **Lưu ý:** Trên macOS, thay `Ctrl` bằng `⌘` (Command).

## Phím tắt Toàn cục

| Phím tắt | Hành động |
|----------|-----------|
| `Ctrl+K` | Mở Quick Connect |
| `Ctrl+Shift+A` | Thêm Host Mới |
| `Ctrl+Shift+T` | Mở Terminal Cục bộ |
| `Ctrl+Shift+L` | Bật/Tắt LAN Discovery |
| `Ctrl+Shift+O` | Chuyển đổi chế độ Terminal/SFTP |

## Phím tắt Terminal

| Phím tắt | Hành động |
|----------|-----------|
| `Ctrl+C` | Sao chép (khi chọn văn bản) / Hủy lệnh |
| `Ctrl+V` | Dán |
| `Ctrl+Shift+C` | Sao chép (thay thế) |
| `Ctrl+Shift+V` | Dán (thay thế) |
| `Ctrl+Shift+F` | Tìm kiếm trong terminal |
| `Ctrl+L` | Xóa màn hình |
| `Ctrl++` | Phóng to |
| `Ctrl+-` | Thu nhỏ |
| `Ctrl+0` | Đặt lại zoom |

## Phím tắt SFTP

| Phím tắt | Hành động |
|----------|-----------|
| `Ctrl+Shift+O` | Chuyển sang SFTP |
| `F2` | Đổi tên file đã chọn |
| `Delete` | Xóa file đã chọn |
| `Ctrl+C` | Sao chép đường dẫn file |
| `Ctrl+R` | Làm mới thư mục |
| `Backspace` | Đi đến thư mục cha |

## Phím tắt Trình soạn thảo

| Phím tắt | Hành động |
|----------|-----------|
| `Ctrl+S` | Lưu file |
| `Ctrl+Z` | Hoàn tác |
| `Ctrl+Shift+Z` | Làm lại |
| `Ctrl+F` | Tìm kiếm |
| `Ctrl+H` | Tìm và thay thế |
| `Ctrl+G` | Đi đến dòng |

## Phím tắt Cửa sổ

| Phím tắt | Hành động |
|----------|-----------|
| `F11` | Bật/Tắt toàn màn hình |
| `Ctrl+Shift+D` | Chuyển đổi chế độ tối/sáng |
| `Ctrl+Q` | Thoát ứng dụng |

## Tùy chỉnh

Hiện tại, phím tắt không thể tùy chỉnh. Tính năng này có thể được thêm vào trong các phiên bản tương lai.

## Mẹo

- Hầu hết phím tắt hoạt động trên tất cả các chế độ xem
- Phím tắt terminal có thể xung đột với phím tắt shell
- Sử dụng `Ctrl+Shift+C/V` trong terminal để tránh xung đột với `Ctrl+C` (hủy)

# Hướng dẫn Bắt đầu Nhanh

Kết nối với máy chủ đầu tiên của bạn trong vòng 5 phút.

## Thêm Host Đầu tiên

1. Nhấp **add new host** trong danh sách máy chủ 
2. Nhập chi tiết máy chủ:
   - **Name**: Bất kỳ tên thân thiện nào (ví dụ: "My VPS")
   - **Protocol**: SSH (mặc định)
   - **Host**: Địa chỉ IP hoặc tên miền
   - **Port**: 22 (mặc định cho SSH)
   - **Username**: Tên người dùng máy chủ của bạn
   - **Authentication**: Password hoặc SSH Key
3. Nhấp **Save**


## Kết nối với Máy chủ

1. Nhấp vào tên host trong danh sách máy chủ
2. Bạn đã kết nối!

## Mẹo Nhanh

### Phím Tắt Terminal

| Phím tắt | Hành động |
|----------|-----------|
| `Ctrl+Shift+C` | Sao chép |
| `Ctrl+Shift+V` | Dán |
| `Ctrl+Shift+F` | Tìm kiếm |
| `Ctrl+L` | Xóa màn hình |
| `Ctrl+Plus` | Phóng to |
| `Ctrl+Minus` | Thu nhỏ |

### Hỗ trợ Đa Tab

- **Tab Mới**: Nhấp + 

### Sao chép-Dán 

3. **Sao chép văn bản**: 
   - Chọn văn bản bằng chuột → Copy
   - Hoặc sử dụng `Ctrl+C`
4. **Dán văn bản**: 
   - Nhấp chuột phải → Paste
   - Hoặc sử dụng `Ctrl+V`

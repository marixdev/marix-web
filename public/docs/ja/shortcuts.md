# キーボードショートカット

Marix のキーボードショートカット完全リスト。

## 表記

| 記号 | キー |
|--------|-----|
| `Ctrl` | Control（Windows/Linux） |
| `⌘` | Command（macOS） |
| `Shift` | Shift |
| `Alt` | Alt/Option |

> **注意:** macOS では、`Ctrl` を `⌘`（Command）に置き換えてください。

## グローバルショートカット

| ショートカット | アクション |
|----------|--------|
| `Ctrl+K` | クイック接続を開く |
| `Ctrl+Shift+A` | 新規ホストを追加 |
| `Ctrl+Shift+T` | ローカルターミナルを開く |
| `Ctrl+Shift+L` | LAN Discovery を切り替え |
| `Ctrl+Shift+O` | ターミナル/SFTP ビューを切り替え |

## ターミナルショートカット

| ショートカット | アクション |
|----------|--------|
| `Ctrl+C` | コピー（テキスト選択時）/ コマンドキャンセル |
| `Ctrl+V` | ペースト |
| `Ctrl+Shift+C` | コピー（代替） |
| `Ctrl+Shift+V` | ペースト（代替） |
| `Ctrl+Shift+F` | ターミナル内検索 |
| `Ctrl+L` | 画面クリア |
| `Ctrl++` | ズームイン |
| `Ctrl+-` | ズームアウト |
| `Ctrl+0` | ズームリセット |

## SFTP ショートカット

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+O` | SFTP に切り替え |
| `F2` | 選択したファイルの名前を変更 |
| `Delete` | 選択したファイルを削除 |
| `Ctrl+C` | ファイルパスをコピー |
| `Ctrl+R` | ディレクトリを更新 |
| `Backspace` | 親ディレクトリへ移動 |

## ファイルエディターショートカット

| ショートカット | アクション |
|----------|--------|
| `Ctrl+S` | ファイルを保存 |
| `Ctrl+Z` | 元に戻す |
| `Ctrl+Shift+Z` | やり直す |
| `Ctrl+F` | 検索 |
| `Ctrl+H` | 検索と置換 |
| `Ctrl+G` | 行へ移動 |

## ウィンドウショートカット

| ショートカット | アクション |
|----------|--------|
| `F11` | フルスクリーン切り替え |
| `Ctrl+Shift+D` | ダーク/ライトモード切り替え |
| `Ctrl+Q` | アプリケーション終了 |

## カスタマイズ

現在、キーボードショートカットはカスタマイズできません。この機能は将来のバージョンで追加される可能性があります。

## ヒント

- ほとんどのショートカットはすべてのビューで動作
- ターミナルショートカットはシェルショートカットと競合する可能性あり
- `Ctrl+C`（キャンセル）との競合を避けるため、ターミナルでは `Ctrl+Shift+C/V` を使用

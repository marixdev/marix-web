## Marix とは？

Marix は、認証情報を**100% ローカル**に保存する、最新かつ安全な SSH クライアントです。クラウドなし、トラッキングなし、妥協なし。

**主な機能:**
- 🔐 **ゼロ知識** - すべてのデータは Argon2id + AES-256 暗号化でローカルに保存
- 🖥️ **マルチプロトコル** - SSH、SFTP、FTP/FTPS、RDP、WebSocket
- 🌐 **クロスプラットフォーム** - Windows、macOS、Linux
- 🌍 **14言語対応** - グローバルユーザー向けに構築
- 🔧 **内蔵ツール** - DNS ルックアップ、WHOIS、ポートスキャナー、Cloudflare DNS マネージャー

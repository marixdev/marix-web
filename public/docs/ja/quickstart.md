# クイックスタートガイド

5分以内に最初のサーバーに接続しましょう。

## 最初のホストを追加

1. サーバーリストの **add new host** をクリック
2. サーバー詳細を入力:
   - **名前**: 任意のわかりやすい名前（例: 「My VPS」）
   - **プロトコル**: SSH（デフォルト）
   - **ホスト**: IP アドレスまたはドメイン
   - **ポート**: 22（SSH のデフォルト）
   - **ユーザー名**: サーバーのユーザー名
   - **認証**: パスワードまたは SSH Key
3. **Save** をクリック


## サーバーに接続

1. サーバーリストでホスト名をクリック
2. 接続完了！

## クイックヒント

### ターミナルショートカット

| ショートカット | アクション |
|----------|--------|
| `Ctrl+Shift+C` | コピー |
| `Ctrl+Shift+V` | ペースト |
| `Ctrl+Shift+F` | 検索 |
| `Ctrl+L` | 画面クリア |
| `Ctrl+Plus` | ズームイン |
| `Ctrl+Minus` | ズームアウト |

### マルチタブサポート

- **新規タブ**: + をクリック

### コピー＆ペースト

3. **テキストをコピー**: 
   - マウスでテキストを選択 → コピー
   - または `Ctrl+C` を使用
4. **テキストをペースト**: 
   - 右クリック → ペースト
   - または `Ctrl+V` を使用

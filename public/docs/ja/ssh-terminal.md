# SSH ターミナル

Marix の SSH ターミナルは、リモートサーバーに接続するための強力で機能豊富なターミナルエミュレーターを提供します。

## ターミナルインターフェース

![Terminal Interface](/terminal.png)

SSH 接続後、以下が表示されます:
- フル機能のターミナルエミュレーター（xterm.js 搭載）
- 上部にターミナル/SFTP ビュー切り替えボタン
- テーマ選択ドロップダウン
- 再接続ボタン

## ターミナルコントロール

| コントロール | 機能 |
|---------|----------|
| ターミナルボタン | ターミナルビューに切り替え |
| SFTP ボタン | ファイルブラウザに切り替え |
| テーマセレクター | ターミナルの配色を変更 |
| ↺ 再接続 | 切断時に再接続 |

## ターミナルの使用

1. 通常の SSH ターミナルと同様に**コマンドを入力**
2. コマンドはリモートサーバーで実行されます
3. **テキストをコピー**: 
   - マウスでテキストを選択 → コピー
   - または `Ctrl+C` を使用
4. **テキストをペースト**: 
   - 右クリック → ペースト
   - または `Ctrl+V` を使用

## キーボードショートカット

| ショートカット | アクション |
|----------|--------|
| `Ctrl+C` | 選択したテキストをコピー |
| `Ctrl+V` | クリップボードからペースト |
| `Ctrl+L` | 画面クリア |
| `Tab` | コマンド履歴を開く/閉じる |

## ターミナルテーマの変更

1. ターミナルセッション中
2. テーマセレクタードロップダウン（ターミナル右上）をクリック
3. 利用可能なテーマを閲覧（400以上のオプション）
4. テーマをクリックして即座に適用

利用可能なテーマカテゴリ:
- クラシック（Tango、Solarized、Monokai）
- モダン（Dracula、One Dark、Nord）
- ハイコントラスト
- カスタムカラー


## ローカルターミナルを開く

リモートサーバーに接続せずにローカルマシンでターミナルを開く:

- キーボードショートカット: `Ctrl+Shift+T`
- ローカルコマンドの実行に便利


## 接続ステータス

タブには接続ステータスが表示されます:
- **緑のインジケーター**: 接続中
- **赤のインジケーター**: 切断
- **黄色のインジケーター**: 接続中

## 再接続

接続が切断された場合:
Marix は SSH を自動再接続します

再接続後もターミナルのスクロール履歴は保持されます。

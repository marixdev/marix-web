# คู่มือเริ่มต้นอย่างรวดเร็ว

เชื่อมต่อกับเซิร์ฟเวอร์แรกของคุณภายใน 5 นาที

## เพิ่มโฮสต์แรกของคุณ

1. คลิก **เพิ่มโฮสต์ใหม่** ในรายการเซิร์ฟเวอร์
2. กรอกรายละเอียดเซิร์ฟเวอร์:
   - **ชื่อ**: ชื่อที่ใช้งานง่าย (เช่น "My VPS")
   - **โปรโตคอล**: SSH (ค่าเริ่มต้น)
   - **โฮสต์**: ที่อยู่ IP หรือโดเมน
   - **พอร์ต**: 22 (ค่าเริ่มต้นสำหรับ SSH)
   - **ชื่อผู้ใช้**: ชื่อผู้ใช้เซิร์ฟเวอร์ของคุณ
   - **การยืนยันตัวตน**: รหัสผ่านหรือ SSH Key
3. คลิก **บันทึก**


## เชื่อมต่อกับเซิร์ฟเวอร์ของคุณ

1. คลิกที่ชื่อโฮสต์ในรายการเซิร์ฟเวอร์
2. คุณเชื่อมต่อแล้ว!

## เคล็ดลับด่วน

### แป้นพิมพ์ลัดเทอร์มินัล

| แป้นพิมพ์ลัด | การทำงาน |
|----------|--------|
| `Ctrl+Shift+C` | คัดลอก |
| `Ctrl+Shift+V` | วาง |
| `Ctrl+Shift+F` | ค้นหา |
| `Ctrl+L` | ล้างหน้าจอ |
| `Ctrl+Plus` | ซูมเข้า |
| `Ctrl+Minus` | ซูมออก |

### รองรับหลายแท็บ

- **แท็บใหม่**: คลิก +

### คัดลอก-วาง

3. **คัดลอกข้อความ**:
   - เลือกข้อความด้วยเมาส์ → คัดลอก
   - หรือใช้ `Ctrl+C`
4. **วางข้อความ**:
   - คลิกขวา → วาง
   - หรือใช้ `Ctrl+V`

# การยืนยันตัวตนสองขั้นตอน (2FA/TOTP)

ตัวสร้างรหัส TOTP ในตัว เหมือน Google Authenticator เก็บรหัสลับ 2FA ของคุณและรับรหัสโดยไม่ต้องใช้โทรศัพท์

## เปิด 2FA

คลิก **"2FA"** ในแถบด้านข้าง

## เพิ่มบัญชี 2FA ใหม่

### ขั้นตอนที่ 1: เปิดกล่องโต้ตอบ
คลิกปุ่ม **"เพิ่มบัญชี"**

### ขั้นตอนที่ 2: กรอกรายละเอียด

| ฟิลด์ | คำอธิบาย | จำเป็น |
|-------|-------------|----------|
| ชื่อบัญชี | ป้ายกำกับสำหรับบัญชีนี้ (เช่น "GitHub", "AWS") | ไม่ (ตั้งชื่ออัตโนมัติ) |
| รหัสลับ | รหัสลับ TOTP (จากหน้าตั้งค่า QR code) | ใช่ |

### ขั้นตอนที่ 3: บันทึก
คลิก **"เพิ่ม"**

## วิธีรับรหัสลับ

เมื่อเปิดใช้งาน 2FA บนเว็บไซต์:

1. มองหา **"สแกน QR code ไม่ได้?"** หรือ **"กรอกด้วยตนเอง"**
2. คัดลอกรหัสลับ (มักเป็นตัวอักษรพิมพ์ใหญ่และตัวเลข)
3. วางใน Marix

ตัวอย่างรูปแบบรหัสลับ:
```
JBSWY3DPEHPK3PXP
```

## การใช้รหัส 2FA

1. ค้นหาบัญชีของคุณในรายการ
2. รหัส 6 หลักแสดงอยู่ข้างๆ
3. ตัวนับเวลาแสดงวินาทีจนกว่ารหัสจะรีเฟรช
4. คลิกปุ่ม **คัดลอก** เพื่อคัดลอกรหัสไปยังคลิปบอร์ด

### ตัวจับเวลา
- รหัสรีเฟรชทุก **30 วินาที**
- ตัวบ่งชี้ความคืบหน้าวงกลมแสดงเวลาที่เหลือ
- รอรหัสใหม่ถ้ารหัสปัจจุบันใกล้หมดอายุ

## แก้ไขชื่อบัญชี

1. คลิกปุ่ม **แก้ไข** ข้างบัญชี
2. กรอกชื่อใหม่
3. กด Enter หรือคลิกบันทึก

## ลบบัญชี 2FA

1. คลิกปุ่ม **ลบ** (ไอคอนถังขยะ)
2. ยืนยันการลบ

> ⚠️ **คำเตือน:** การลบจะลบรหัสลับ ตรวจสอบให้แน่ใจว่าคุณมีวิธีเข้าถึงสำรองของบัญชีที่เชื่อมโยงก่อนลบ

## บริการทั่วไป

นี่คือบริการยอดนิยมบางส่วนที่รองรับ TOTP:

| บริการ | ตำแหน่งการตั้งค่า 2FA |
|---------|----------------------|
| GitHub | Settings → Password and authentication |
| AWS | IAM → Security credentials |
| Google | Google Account → Security |
| Microsoft | Security settings → Two-step verification |
| Cloudflare | Profile → Authentication |
| DigitalOcean | Account → Security |
| Linode | My Profile → Password & Authentication |

## แนวปฏิบัติที่ดี

1. **สำรองรหัสลับของคุณ** - เก็บรหัสลับอย่างปลอดภัย
2. **เก็บรหัสกู้คืน** - บันทึกรหัสสำรองที่บริการให้มา
3. **ใช้ชื่อที่อธิบายได้ชัดเจน** - ระบุบัญชีได้ง่ายและรวดเร็ว
4. **ทดสอบรหัสก่อนพึ่งพา** - ตรวจสอบว่าใช้งานได้ทันทีหลังเพิ่ม
5. **ซิงค์เวลา** - ตรวจสอบให้แน่ใจว่านาฬิกาคอมพิวเตอร์ของคุณถูกต้อง

## การแก้ไขปัญหา

### รหัสไม่ทำงาน

1. **ตรวจสอบการซิงค์เวลา** - นาฬิกาคอมพิวเตอร์ของคุณต้องถูกต้อง
2. **ตรวจสอบรหัสลับ** - กรอกใหม่หากคัดลอกผิด
3. **รอรหัสถัดไป** - รหัสปัจจุบันอาจกำลังหมดอายุ

### สูญเสียการเข้าถึง

หากคุณสูญเสียการเข้าถึง Marix:
1. ใช้ **รหัสกู้คืน** ที่บริการให้มา
2. ติดต่อฝ่ายสนับสนุนบริการเพื่อกู้คืนบัญชี
3. นี่คือเหตุผลที่การสำรองรหัสลับสำคัญ!

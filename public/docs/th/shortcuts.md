# ปุ่มลัด

ปุ่มลัดแป้นพิมพ์สำหรับเพิ่มประสิทธิภาพการใช้งาน

## หน้าจอปุ่มลัด

![หน้าจอปุ่มลัด](/shortcuts.png)

## ดูปุ่มลัดทั้งหมด

กด **Ctrl+K** (Windows/Linux) หรือ **⌘+K** (macOS) เพื่อเปิดหน้าจอปุ่มลัด

## ปุ่มลัดทั่วไป

| ปุ่มลัด | การกระทำ |
|----------|--------|
| Ctrl+N | สร้างการเชื่อมต่อใหม่ |
| Ctrl+K | เปิดหน้าจอปุ่มลัด |
| Ctrl+, | เปิดการตั้งค่า |
| Ctrl+Q | ปิดแอปพลิเคชัน |
| F11 | สลับโหมดเต็มจอ |
| Ctrl+Tab | สลับแท็บถัดไป |
| Ctrl+Shift+Tab | สลับแท็บก่อนหน้า |
| Ctrl+W | ปิดแท็บปัจจุบัน |

## กำหนดปุ่มลัดเอง

### ขั้นตอนที่ 1: เปิดการตั้งค่า
ไปที่ **การตั้งค่า** → **ปุ่มลัด**

### ขั้นตอนที่ 2: เลือกการกระทำ
คลิกการกระทำที่ต้องการกำหนดปุ่มลัดใหม่

### ขั้นตอนที่ 3: กดปุ่มลัดใหม่
กดปุ่มลัดที่ต้องการ

### ขั้นตอนที่ 4: บันทึก
คลิก **"บันทึก"**

## รีเซ็ตปุ่มลัด

1. ไปที่ **การตั้งค่า** → **ปุ่มลัด**
2. คลิก **"รีเซ็ตเป็นค่าเริ่มต้น"**
3. ยืนยันการรีเซ็ต

## เคล็ดลับ

1. **เรียนรู้ปุ่มลัดทีละน้อย** - เริ่มจากที่ใช้บ่อยที่สุด
2. **ใช้ Ctrl+K** เพื่อดูปุ่มลัดทั้งหมด
3. **กำหนดปุ่มลัดเอง** สำหรับการกระทำที่ใช้บ่อย
4. **ฝึกใช้ประจำ** จนเป็นนิสัย

> หมายเหตุ: บน macOS ให้แทนที่ Ctrl ด้วย ⌘ (Command)

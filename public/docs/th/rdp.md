# Remote Desktop (RDP)

เชื่อมต่อกับเดสก์ท็อป Windows ระยะไกลโดยใช้ Remote Desktop Protocol

## หน้าจอ RDP

![หน้าจอ RDP](/rdp.png)

## รองรับแพลตฟอร์ม

- **Windows** - เซิร์ฟเวอร์ Windows ทุกเวอร์ชันที่มี RDP
- **Linux** - เซิร์ฟเวอร์ที่เปิดใช้ xrdp
- **macOS** - ผ่านซอฟต์แวร์ RDP ของบุคคลที่สาม

## เพิ่มการเชื่อมต่อ RDP

### ขั้นตอนที่ 1: เปิดหน้าจอ RDP
คลิก **"RDP"** ในเมนูแถบด้านข้าง

### ขั้นตอนที่ 2: เพิ่มการเชื่อมต่อใหม่
คลิกปุ่ม **"เพิ่ม"**

### ขั้นตอนที่ 3: กรอกรายละเอียดการเชื่อมต่อ

| ฟิลด์ | คำอธิบาย | ตัวอย่าง |
|-------|-------------|---------|
| ชื่อ | ป้ายบอกการเชื่อมต่อ | "Windows Server" |
| โฮสต์ | ที่อยู่ IP หรือชื่อโฮสต์ | 192.168.1.100 |
| พอร์ต | พอร์ต RDP (ค่าเริ่มต้น: 3389) | 3389 |
| ชื่อผู้ใช้ | ชื่อผู้ใช้ Windows | Administrator |
| รหัสผ่าน | รหัสผ่าน Windows | ••••••••• |
| โดเมน | โดเมน Windows (ไม่บังคับ) | WORKGROUP |

### ขั้นตอนที่ 4: บันทึก
คลิก **"บันทึก"**

## ตั้งค่าขั้นสูง

### การตั้งค่าการแสดงผล

| ตัวเลือก | คำอธิบาย |
|--------|-------------|
| ความละเอียด | ขนาดหน้าจอ (เช่น 1920x1080) |
| ความลึกสี | 16-bit, 24-bit หรือ 32-bit |
| เต็มจอ | เปิดใช้โหมดเต็มจอ |

### การตั้งค่าประสิทธิภาพ

| ตัวเลือก | คำอธิบาย |
|--------|-------------|
| ปิดใช้วอลล์เปเปอร์ | ลดแบนด์วิดท์ |
| ปิดใช้ธีม | ประสิทธิภาพดีขึ้น |
| ปิดใช้การลากหน้าต่างเต็มรูป | การเลื่อนราบรื่นขึ้น |
| ปิดใช้แอนิเมชันเมนู | โต้ตอบเร็วขึ้น |

### การแชร์ทรัพยากร

| ตัวเลือก | คำอธิบาย |
|--------|-------------|
| แชร์คลิปบอร์ด | คัดลอก/วางระหว่างเครื่อง |
| แชร์เสียง | เล่นเสียงระยะไกลในเครื่อง |
| แชร์ไดรฟ์ | เข้าถึงไดรฟ์ในเครื่องจากระยะไกล |

## เชื่อมต่อ

1. ค้นหาการเชื่อมต่อในรายการ
2. ดับเบิลคลิก หรือคลิก **"เชื่อมต่อ"**
3. รอการสร้างเซสชัน
4. ใช้เดสก์ท็อป Windows ระยะไกล

## การควบคุมเซสชัน

### ระหว่างเซสชัน
- **เต็มจอ:** กด F11 หรือคลิกไอคอนเต็มจอ
- **คลิปบอร์ด:** คัดลอก/วางทำงานตามปกติถ้าเปิดใช้
- **แป้นพิเศษ:** Ctrl+Alt+Del มีปุ่มเฉพาะ

### ยกเลิกการเชื่อมต่อ
- คลิกปุ่ม **"ยกเลิกการเชื่อมต่อ"**
- หรือปิดแท็บเซสชัน
- Log off จาก Windows เพื่อจบเซสชัน

## RDP ผ่าน SSH Tunnel

สำหรับการเชื่อมต่อที่ปลอดภัย ใช้ SSH port forwarding:

1. สร้าง local port forward:
   - พอร์ตในเครื่อง: 13389
   - Remote Host: windows-server-ip
   - Remote Port: 3389

2. เชื่อมต่อ RDP กับ:
   - โฮสต์: localhost
   - พอร์ต: 13389

## แก้ปัญหา

### การเชื่อมต่อถูกปฏิเสธ
- ตรวจสอบว่า Remote Desktop เปิดใช้บนเซิร์ฟเวอร์
- ยืนยันว่า firewall อนุญาตพอร์ต 3389
- ตรวจสอบ IP และพอร์ต

### การยืนยันตัวตนล้มเหลว
- ยืนยันข้อมูลประจำตัว
- ตรวจสอบการตั้งค่าโดเมน
- ลอง format: `DOMAIN\username` หรือ `username@domain`

### หน้าจอดำ
- รอสักครู่ให้เซสชันโหลด
- ลดความละเอียด/ความลึกสี
- ปิดใช้การเร่งความเร็วฮาร์ดแวร์

### ประสิทธิภาพช้า
- ลดความละเอียดหน้าจอ
- เปิดใช้การปรับปรุงประสิทธิภาพ (ปิดวอลล์เปเปอร์, ฯลฯ)
- ตรวจสอบแบนด์วิดท์เครือข่าย
- ใช้ SSH tunnel สำหรับการเชื่อมต่อที่เสถียรกว่า

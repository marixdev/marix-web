# แชร์เซิร์ฟเวอร์ผ่าน LAN

แชร์การเชื่อมต่อเซิร์ฟเวอร์กับอุปกรณ์อื่นบนเครือข่ายท้องถิ่น

## หน้าจอแชร์เซิร์ฟเวอร์ LAN

![หน้าจอแชร์เซิร์ฟเวอร์ LAN](/lanshare.png)

## ภาพรวม

ฟีเจอร์แชร์เซิร์ฟเวอร์ LAN ช่วยให้คุณส่งรายละเอียดการเชื่อมต่อเซิร์ฟเวอร์ไปยังอุปกรณ์อื่นบนเครือข่ายท้องถิ่นเดียวกัน ทำให้การตั้งค่าเซิร์ฟเวอร์เหมือนกันบนหลายอุปกรณ์ทำได้ง่าย

## ความต้องการ

- อุปกรณ์ทั้งสองต้องอยู่ในเครือข่ายท้องถิ่นเดียวกัน
- ต้องติดตั้งแอปพลิเคชันบนอุปกรณ์ทั้งสอง
- ทั้งสองอุปกรณ์ต้องเปิดใช้งานการแชร์ LAN

## แชร์เซิร์ฟเวอร์

### ขั้นตอนที่ 1: เลือกเซิร์ฟเวอร์
1. ไปที่รายการเซิร์ฟเวอร์ของคุณ
2. คลิกขวาที่เซิร์ฟเวอร์ที่ต้องการแชร์
3. เลือก **"แชร์ผ่าน LAN"**

### ขั้นตอนที่ 2: เลือกอุปกรณ์ปลายทาง
1. รายการอุปกรณ์ที่พร้อมใช้งานปรากฏขึ้น
2. คลิกอุปกรณ์ที่ต้องการส่งการเชื่อมต่อ

### ขั้นตอนที่ 3: ส่ง
1. คลิก **"แชร์"**
2. รอการยอมรับจากอุปกรณ์ปลายทาง

## รับการแชร์เซิร์ฟเวอร์

### ขั้นตอนที่ 1: รอการแจ้งเตือน
เมื่ออุปกรณ์อื่นแชร์เซิร์ฟเวอร์ คุณจะเห็นการแจ้งเตือน

### ขั้นตอนที่ 2: ตรวจสอบและยอมรับ
1. ดูรายละเอียดเซิร์ฟเวอร์ที่แชร์
2. คลิก **"ยอมรับ"** เพื่อเพิ่มในรายการของคุณ
3. หรือคลิก **"ปฏิเสธ"** เพื่อยกเลิก

### ขั้นตอนที่ 3: ปรับแต่ง (ไม่บังคับ)
หลังจากยอมรับ คุณสามารถแก้ไขรายละเอียดเซิร์ฟเวอร์ได้ตามต้องการ

## ข้อมูลที่แชร์

| รายการ | รวมอยู่ |
|------|----------|
| ชื่อโฮสต์/IP | ✅ ใช่ |
| พอร์ต | ✅ ใช่ |
| ชื่อผู้ใช้ | ✅ ใช่ |
| รหัสผ่าน | ⚙️ ไม่บังคับ |
| SSH Key | ⚙️ ไม่บังคับ |
| ป้ายกำกับ/กลุ่ม | ✅ ใช่ |

> ⚠️ **หมายเหตุความปลอดภัย:** รหัสผ่านและ SSH key จะแชร์เฉพาะเมื่อคุณเลือกเปิดใช้ตัวเลือกนี้

## ตั้งค่าความปลอดภัย

### การแชร์ข้อมูลประจำตัว

| ตัวเลือก | คำอธิบาย |
|--------|-------------|
| แชร์รหัสผ่าน | รวมรหัสผ่านในการแชร์ |
| แชร์ SSH Key | รวม SSH key ในการแชร์ |
| ต้องการการยืนยัน | ผู้รับต้องกรอกข้อมูลประจำตัวเอง |

## แชร์หลายเซิร์ฟเวอร์

1. เลือกหลายเซิร์ฟเวอร์ (กด Ctrl/Cmd ค้างไว้แล้วคลิก)
2. คลิกขวา → **"แชร์ที่เลือกผ่าน LAN"**
3. เลือกอุปกรณ์ปลายทาง
4. คลิก **"แชร์"**

## แก้ปัญหา

### ไม่เห็นอุปกรณ์อื่น
- ตรวจสอบว่าทั้งสองอุปกรณ์อยู่ในเครือข่ายเดียวกัน
- ยืนยันว่าการแชร์ LAN เปิดใช้
- ตรวจสอบ firewall

### การแชร์ล้มเหลว
- ยืนยันการเชื่อมต่อเครือข่าย
- ลองปิดและเปิดการแชร์ LAN ใหม่
- รีสตาร์ทแอปพลิเคชันบนอุปกรณ์ทั้งสอง

### เซิร์ฟเวอร์ที่รับไม่ทำงาน
- ตรวจสอบว่าข้อมูลเซิร์ฟเวอร์ถูกต้อง
- ยืนยันการเชื่อมต่อเครือข่ายไปยังเซิร์ฟเวอร์
- กรอกข้อมูลประจำตัวถ้าไม่ได้รวมอยู่

## เคล็ดลับ

1. **ตรวจสอบก่อนแชร์** ว่าข้อมูลเซิร์ฟเวอร์ถูกต้อง
2. **พิจารณาความปลอดภัย** ก่อนแชร์ข้อมูลประจำตัว
3. **ใช้สำหรับการตั้งค่าเริ่มต้น** แล้วให้ผู้รับเปลี่ยนรหัสผ่าน
4. **แชร์ตามกลุ่ม** เพื่อประหยัดเวลา

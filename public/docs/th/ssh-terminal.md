# เทอร์มินัล SSH

เทอร์มินัล SSH ใน Marix มีเทอร์มินัลอีมูเลเตอร์ที่ทรงพลังและครบถ้วนสำหรับเชื่อมต่อกับเซิร์ฟเวอร์ระยะไกล

## หน้าจอเทอร์มินัล

![หน้าจอเทอร์มินัล](/terminal.png)

หลังเชื่อมต่อผ่าน SSH คุณจะเห็น:
- เทอร์มินัลอีมูเลเตอร์เต็มรูปแบบ (ขับเคลื่อนโดย xterm.js)
- ปุ่มสลับสำหรับมุมมอง Terminal/SFTP ด้านบน
- เมนูดร็อปดาวน์เลือกธีม
- ปุ่มเชื่อมต่อใหม่

## การควบคุมเทอร์มินัล

| การควบคุม | ฟังก์ชัน |
|---------|----------|
| ปุ่ม Terminal | สลับไปมุมมองเทอร์มินัล |
| ปุ่ม SFTP | สลับไปเรียกดูไฟล์ |
| เลือกธีม | เปลี่ยนสีเทอร์มินัล |
| ↺ เชื่อมต่อใหม่ | เชื่อมต่อใหม่ถ้าขาดการเชื่อมต่อ |

## การใช้เทอร์มินัล

1. **พิมพ์คำสั่ง** เหมือนเทอร์มินัล SSH ปกติ
2. คำสั่งทำงานบนเซิร์ฟเวอร์ระยะไกล
3. **คัดลอกข้อความ**:
   - เลือกข้อความด้วยเมาส์ → คัดลอก
   - หรือใช้ `Ctrl+C`
4. **วางข้อความ**:
   - คลิกขวา → วาง
   - หรือใช้ `Ctrl+V`

## แป้นพิมพ์ลัด

| แป้นพิมพ์ลัด | การทำงาน |
|----------|--------|
| `Ctrl+C` | คัดลอกข้อความที่เลือก |
| `Ctrl+V` | วางจากคลิปบอร์ด |
| `Ctrl+L` | ล้างหน้าจอ |
| `Tab` | เปิด/ปิดประวัติคำสั่ง |

## เปลี่ยนธีมเทอร์มินัล

1. ขณะอยู่ในเซสชันเทอร์มินัล
2. คลิกเมนูดร็อปดาวน์เลือกธีม (ด้านบนขวาของเทอร์มินัล)
3. เรียกดูธีมที่มี (400+ ตัวเลือก)
4. คลิกธีมเพื่อใช้งานทันที

หมวดหมู่ธีมที่มี:
- Classic (Tango, Solarized, Monokai)
- Modern (Dracula, One Dark, Nord)
- High Contrast
- สีที่กำหนดเอง


## เปิดเทอร์มินัลในเครื่อง

เปิดเทอร์มินัลบนเครื่องของคุณโดยไม่ต้องเชื่อมต่อกับเซิร์ฟเวอร์ระยะไกล:

- แป้นพิมพ์ลัด: `Ctrl+Shift+T`
- มีประโยชน์สำหรับรันคำสั่งในเครื่อง


## สถานะการเชื่อมต่อ

แท็บแสดงสถานะการเชื่อมต่อ:
- **ตัวบ่งชี้สีเขียว**: เชื่อมต่อแล้ว
- **ตัวบ่งชี้สีแดง**: ขาดการเชื่อมต่อ
- **ตัวบ่งชี้สีเหลือง**: กำลังเชื่อมต่อ

## การเชื่อมต่อใหม่

ถ้าการเชื่อมต่อขาด:
Marix เชื่อมต่อ ssh ใหม่อัตโนมัติ

เทอร์มินัลเก็บประวัติเลื่อนดูหลังเชื่อมต่อใหม่

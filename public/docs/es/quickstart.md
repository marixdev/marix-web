# Guía de Inicio Rápido

Conéctate a tu primer servidor en menos de 5 minutos.

## Añade Tu Primer Host

1. Haz clic en **añadir nuevo host** en la lista de servidores 
2. Ingresa los detalles del servidor:
   - **Nombre**: Cualquier nombre descriptivo (ej., "Mi VPS")
   - **Protocolo**: SSH (predeterminado)
   - **Host**: Dirección IP o dominio
   - **Puerto**: 22 (predeterminado para SSH)
   - **Usuario**: Tu nombre de usuario del servidor
   - **Autenticación**: Contraseña o Clave SSH
3. Haz clic en **Guardar**


## Conéctate a Tu Servidor

1. Haz clic en el nombre del host en la lista de servidores
2. ¡Ya estás conectado!

## Consejos Rápidos

### Atajos del Terminal

| Atajo | Acción |
|-------|--------|
| `Ctrl+Shift+C` | Copiar |
| `Ctrl+Shift+V` | Pegar |
| `Ctrl+Shift+F` | Buscar |
| `Ctrl+L` | Limpiar pantalla |
| `Ctrl+Plus` | Acercar |
| `Ctrl+Minus` | Alejar |

### Soporte Multi-Pestaña

- **Nueva Pestaña**: Haz clic en + 

### Copiar-Pegar 

3. **Copiar texto**: 
   - Selecciona texto con el ratón → Copiar
   - O usa `Ctrl+C`
4. **Pegar texto**: 
   - Clic derecho → Pegar
   - O usa `Ctrl+V`

# Atajos de Teclado

Referencia rápida de atajos de teclado disponibles en la terminal de Marix.

## Atajos de Terminal

| Atajo | Acción |
|-------|--------|
| `Ctrl+C` | Cancelar comando actual |
| `Ctrl+D` | Cerrar sesión (EOF) |
| `Ctrl+Z` | Suspender proceso actual |
| `Ctrl+L` | Limpiar pantalla |
| `Ctrl+A` | Mover cursor al inicio de línea |
| `Ctrl+E` | Mover cursor al final de línea |
| `Ctrl+U` | Eliminar desde cursor hasta inicio de línea |
| `Ctrl+K` | Eliminar desde cursor hasta final de línea |
| `Ctrl+W` | Eliminar palabra anterior |
| `Ctrl+R` | Buscar en historial de comandos |
| `Tab` | Autocompletar comando/ruta |
| `↑` / `↓` | Navegar historial de comandos |

## Atajos de Aplicación

| Atajo | Acción |
|-------|--------|
| `Ctrl+T` | Nueva pestaña de terminal |
| `Ctrl+W` | Cerrar pestaña actual |
| `Ctrl+Tab` | Siguiente pestaña |
| `Ctrl+Shift+Tab` | Pestaña anterior |
| `Ctrl+1-9` | Ir a pestaña específica |
| `Ctrl+Shift+C` | Copiar selección |
| `Ctrl+Shift+V` | Pegar desde portapapeles |
| `Ctrl+F` | Buscar en terminal |
| `Ctrl++` | Aumentar tamaño de fuente |
| `Ctrl+-` | Reducir tamaño de fuente |
| `Ctrl+0` | Restablecer tamaño de fuente |
| `F11` | Alternar pantalla completa |

## Atajos de Selección

| Atajo | Acción |
|-------|--------|
| `Shift+↑/↓/←/→` | Extender selección |
| `Ctrl+Shift+←/→` | Seleccionar palabra |
| `Triple clic` | Seleccionar línea completa |
| `Ctrl+A` | Seleccionar todo (en algunos modos) |

## Atajos de SFTP

| Atajo | Acción |
|-------|--------|
| `Ctrl+U` | Subir archivo |
| `Ctrl+D` | Descargar seleccionado |
| `Ctrl+N` | Nueva carpeta |
| `F2` | Renombrar seleccionado |
| `Delete` | Eliminar seleccionado |
| `Ctrl+A` | Seleccionar todo |
| `Ctrl+C` | Copiar seleccionado |
| `Ctrl+X` | Cortar seleccionado |
| `Ctrl+V` | Pegar |
| `F5` | Actualizar directorio |
| `Backspace` | Ir al directorio padre |

## Atajos de Editor

Cuando editas archivos en el editor integrado:

| Atajo | Acción |
|-------|--------|
| `Ctrl+S` | Guardar archivo |
| `Ctrl+Z` | Deshacer |
| `Ctrl+Y` | Rehacer |
| `Ctrl+F` | Buscar |
| `Ctrl+H` | Buscar y reemplazar |
| `Ctrl+G` | Ir a línea |
| `Ctrl+/` | Comentar/descomentar línea |
| `Tab` | Indentar |
| `Shift+Tab` | Desindentar |

## Personalizar Atajos

Actualmente, los atajos de teclado son fijos y no se pueden personalizar.
Futuras versiones pueden incluir mapeo de teclas personalizable.

## Notas de Plataforma

- En **macOS**, usa `Cmd` en lugar de `Ctrl`
- En **Linux**, algunos atajos pueden entrar en conflicto con atajos del sistema
- En **Android/iOS**, los atajos de teclado funcionan con teclados externos

# Cloudflare DNS 管理器

直接从 Marix 管理您的 Cloudflare DNS 记录。

## Cloudflare DNS 管理器界面

![Cloudflare DNS Manager Interface](/cf.png)

## 设置

### 步骤 1：获取 API 令牌

1. 访问 [Cloudflare 仪表板](https://dash.cloudflare.com)
2. 导航到 个人资料 → API 令牌
3. 点击"创建令牌"
4. 使用"编辑区域 DNS"模板或创建自定义令牌：
   - Zone.DNS：编辑
   - Zone.Zone：读取
5. 复制令牌

### 步骤 2：在 Marix 中配置

1. 转到侧边栏中的 **设置**
2. 找到 **"Cloudflare API 令牌"** 部分
3. 点击 **"添加令牌"**
4. 粘贴您的令牌
5. 点击 **"保存"**

## 使用 Cloudflare DNS

1. 在侧边栏中点击 **"Cloudflare"**
2. 您的域名加载在下拉菜单中
3. 选择一个域名
4. 该域名的 DNS 记录出现

## 查看 DNS 记录

表格显示：

| 列 | 描述 |
|--------|-------------|
| 类型 | 记录类型（A、AAAA、CNAME、MX、TXT 等） |
| 名称 | 记录名称/子域名 |
| 内容 | 记录值（IP、主机名、文本） |
| TTL | 生存时间（自动或秒数） |
| 代理 | 通过 Cloudflare 代理或仅 DNS |

## 添加 DNS 记录

### 步骤 1：打开对话框
点击 **"添加记录"** 按钮

### 步骤 2：填写详情

| 字段 | 描述 |
|-------|-------------|
| 类型 | 选择：A、AAAA、CNAME、MX、TXT、SRV 等 |
| 名称 | 子域名（例如"www"）或"@"表示根域名 |
| 内容 | 值（A 记录的 IP、CNAME 的主机名等） |
| TTL | 生存时间：自动或自定义秒数 |
| 代理 | 切换：已代理（橙色云）或仅 DNS（灰色） |
| 注释 | 可选备注 |

### 步骤 3：创建
点击 **"创建"**

## 常见记录类型

### A 记录
将域名指向 IPv4 地址。

| 字段 | 示例 |
|-------|---------|
| 类型 | A |
| 名称 | @（或 www） |
| 内容 | 192.168.1.100 |

### AAAA 记录
将域名指向 IPv6 地址。

| 字段 | 示例 |
|-------|---------|
| 类型 | AAAA |
| 名称 | @ |
| 内容 | 2001:db8::1 |

### CNAME 记录
创建到另一个域名的别名。

| 字段 | 示例 |
|-------|---------|
| 类型 | CNAME |
| 名称 | www |
| 内容 | example.com |

### MX 记录
指定邮件服务器。

| 字段 | 示例 |
|-------|---------|
| 类型 | MX |
| 名称 | @ |
| 内容 | mail.example.com |
| 优先级 | 10 |

### TXT 记录
存储文本数据（SPF、DKIM、验证）。

| 字段 | 示例 |
|-------|---------|
| 类型 | TXT |
| 名称 | @ |
| 内容 | v=spf1 include:_spf.google.com ~all |

## 编辑 DNS 记录

1. 在表格中找到记录
2. 点击 **编辑** 按钮（铅笔图标）
3. 修改字段
4. 点击 **"保存"**

## 删除 DNS 记录

1. 找到记录
2. 点击 **删除** 按钮（垃圾桶图标）
3. 确认删除

## 代理状态

| 图标 | 状态 | 效果 |
|------|--------|--------|
| 🟠 橙色云 | 已代理 | 流量通过 Cloudflare CDN |
| ⚪ 灰色云 | 仅 DNS | 直接连接到源 |

### 何时使用代理

**已代理（橙色）：**
- Web 服务器（HTTP/HTTPS）
- 需要 DDoS 防护
- 需要 CDN 缓存

**仅 DNS（灰色）：**
- SSH、FTP、邮件服务器
- 非 HTTP 服务
- 需要直接 IP 连接

## 刷新记录

点击域名下拉菜单旁边的 **刷新** 按钮重新加载记录。

## 提示

- **TTL 自动** 在大多数情况下推荐
- 使用 DNS 查询工具 **测试更改**
- 进行重大更改前 **备份记录**
- **传播** 可能需要全球范围内最多 48 小时

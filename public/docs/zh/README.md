## 什么是 Marix？

Marix 是一款现代、安全的 SSH 客户端，您的凭证 **100% 本地存储**。无云端、无追踪、无妥协。

**核心特性：**
- 🔐 **零知识** - 所有数据使用 Argon2id + AES-256 加密本地存储
- 🖥️ **多协议** - SSH、SFTP、FTP/FTPS、RDP、WebSocket
- 🌐 **跨平台** - Windows、macOS、Linux
- 🌍 **14 种语言** - 为全球用户打造
- 🔧 **内置工具** - DNS 查询、WHOIS、端口扫描、Cloudflare DNS 管理器

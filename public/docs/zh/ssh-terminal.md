# SSH 终端

Marix 中的 SSH 终端提供了功能强大、特性丰富的终端模拟器，用于连接远程服务器。

## 终端界面

![Terminal Interface](/terminal.png)

通过 SSH 连接后，您将看到：
- 完整的终端模拟器（由 xterm.js 驱动）
- 顶部的终端/SFTP 视图切换按钮
- 主题选择下拉菜单
- 重新连接按钮

## 终端控件

| 控件 | 功能 |
|---------|----------|
| 终端按钮 | 切换到终端视图 |
| SFTP 按钮 | 切换到文件浏览器 |
| 主题选择器 | 更改终端配色方案 |
| ↺ 重新连接 | 断开连接后重新连接 |

## 使用终端

1. **输入命令** 就像普通 SSH 终端一样
2. 命令在远程服务器上执行
3. **复制文本**：
   - 用鼠标选择文本 → 复制
   - 或使用 `Ctrl+C`
4. **粘贴文本**：
   - 右键点击 → 粘贴
   - 或使用 `Ctrl+V`

## 键盘快捷键

| 快捷键 | 操作 |
|----------|--------|
| `Ctrl+C` | 复制选中的文本 |
| `Ctrl+V` | 从剪贴板粘贴 |
| `Ctrl+L` | 清屏 |
| `Tab` | 打开/关闭命令历史 |

## 更改终端主题

1. 在终端会话中
2. 点击主题选择器下拉菜单（终端右上角）
3. 浏览可用主题（400+ 选项）
4. 点击主题即可立即应用

可用主题类别：
- 经典（Tango、Solarized、Monokai）
- 现代（Dracula、One Dark、Nord）
- 高对比度
- 自定义颜色


## 打开本地终端

无需连接远程服务器即可在本地机器上打开终端：

- 键盘快捷键：`Ctrl+Shift+T`
- 用于运行本地命令


## 连接状态

标签页显示连接状态：
- **绿色指示器**：已连接
- **红色指示器**：已断开
- **黄色指示器**：连接中

## 重新连接

如果连接断开：
Marix 会自动重新连接 SSH

重新连接后终端会保留滚动历史。

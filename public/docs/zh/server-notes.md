# 服务器备注（便签）

Marix 提供 **服务器备注** 功能，允许您为每个服务器附加个人笔记。这对于记录服务器配置、维护提醒或快速参考信息非常有用。

## 服务器备注界面

![Server Notes Interface](/note.png)

## 功能

- **每服务器备注**：每个服务器可以有自己的备注
- **自动保存**：更改在您输入时自动保存（500ms 延迟）
- **富文本支持**：带换行的纯文本
- **视觉指示器**：有备注的服务器显示填充的备注图标
- **与服务器数据同步**：备注包含在服务器备份中

## 使用方法

### 打开备注

1. 连接到服务器（SSH 终端或 SFTP）
2. 点击顶部工具栏中的 **备注** 按钮（铅笔图标）
3. 便签弹窗出现在右下角

### 编写备注

- 直接在文本区域输入
- 备注在 500ms 无活动后 **自动保存**
- 保存进行时显示"正在保存..."指示器
- 按 `Escape` 或点击外部关闭

### 视觉指示器

| 图标状态 | 含义 |
|------------|---------|
| 轮廓（空） | 此服务器没有备注 |
| 填充（实心） | 服务器有备注 |
| 琥珀色/黄色 | 存在备注 |

## 数据存储

- 备注作为服务器配置的一部分存储
- ✅ **包含在本地备份中**（Marix 备份功能）
- ✅ **包含在云备份中**（Google Drive、Box）
- ✅ 使用备份加密时 **已加密**

## 键盘快捷键

| 按键 | 操作 |
|-----|--------|
| `Escape` | 关闭备注弹窗 |
| 点击外部 | 关闭备注弹窗 |

## 常见问题

### 备注是否加密？

备注在服务器配置文件中以纯文本存储。当您创建加密备份时，备注会与所有服务器数据一起加密。

### 我可以使用 Markdown 吗？

备注目前仅支持纯文本。Markdown 渲染可能在未来版本中添加。

### 有字符限制吗？

没有硬性限制，但非常长的备注可能影响性能。建议保持备注简洁。

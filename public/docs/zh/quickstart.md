# 快速入门指南

5 分钟内连接到您的第一台服务器。

## 添加您的第一台主机

1. 在服务器列表中点击 **添加新主机**
2. 输入服务器详情：
   - **名称**：任意友好名称（例如"我的 VPS"）
   - **协议**：SSH（默认）
   - **主机**：IP 地址或域名
   - **端口**：22（SSH 默认端口）
   - **用户名**：您的服务器用户名
   - **认证方式**：密码或 SSH 密钥
3. 点击 **保存**


## 连接到您的服务器

1. 在服务器列表中点击主机名称
2. 连接完成！

# 键盘快捷键

Marix 中的完整键盘快捷键列表。

## 符号说明

| 符号 | 按键 |
|--------|-----|
| `Ctrl` | Control（Windows/Linux） |
| `⌘` | Command（macOS） |
| `Shift` | Shift |
| `Alt` | Alt/Option |

> **注意：** 在 macOS 上，将 `Ctrl` 替换为 `⌘`（Command）。

## 全局快捷键

| 快捷键 | 操作 |
|----------|--------|
| `Ctrl+K` | 打开快速连接 |
| `Ctrl+Shift+A` | 添加新主机 |
| `Ctrl+Shift+T` | 打开本地终端 |
| `Ctrl+Shift+L` | 切换局域网发现 |
| `Ctrl+Shift+O` | 切换终端/SFTP 视图 |

## 终端快捷键

| 快捷键 | 操作 |
|----------|--------|
| `Ctrl+C` | 复制（选中文本时）/ 取消命令 |
| `Ctrl+V` | 粘贴 |
| `Ctrl+Shift+C` | 复制（替代方式） |
| `Ctrl+Shift+V` | 粘贴（替代方式） |
| `Ctrl+Shift+F` | 在终端中查找 |
| `Ctrl+L` | 清屏 |
| `Ctrl++` | 放大 |
| `Ctrl+-` | 缩小 |
| `Ctrl+0` | 重置缩放 |

## SFTP 快捷键

| 快捷键 | 操作 |
|----------|--------|
| `Ctrl+Shift+O` | 切换到 SFTP |
| `F2` | 重命名选中的文件 |
| `Delete` | 删除选中的文件 |
| `Ctrl+C` | 复制文件路径 |
| `Ctrl+R` | 刷新目录 |
| `Backspace` | 转到上级目录 |

## 文件编辑器快捷键

| 快捷键 | 操作 |
|----------|--------|
| `Ctrl+S` | 保存文件 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Shift+Z` | 重做 |
| `Ctrl+F` | 查找 |
| `Ctrl+H` | 查找和替换 |
| `Ctrl+G` | 跳转到行 |

## 窗口快捷键

| 快捷键 | 操作 |
|----------|--------|
| `F11` | 切换全屏 |
| `Ctrl+Shift+D` | 切换深色/浅色模式 |
| `Ctrl+Q` | 退出应用程序 |

## 自定义

目前无法自定义键盘快捷键。此功能可能在未来版本中添加。

## 提示

- 大多数快捷键在所有视图中都有效
- 终端快捷键可能与 shell 快捷键冲突
- 在终端中使用 `Ctrl+Shift+C/V` 以避免与 `Ctrl+C`（取消）冲突

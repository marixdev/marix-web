# 局域网文件传输

在同一本地网络上的设备之间传输文件，无需服务器。

## 局域网文件传输界面

![LAN File Transfer Interface](/sendfile.png)

## 要求

> ⚠️ **重要：** 发送方和接收方都必须 **启用局域网发现**。
> 
> 点击标题栏中的 🌐 按钮或使用 `Ctrl+Shift+L` 切换。

- 两台设备在同一网络（WiFi/局域网）
- 两台设备都启用局域网发现
- 两台设备都运行 Marix

## 打开局域网文件传输

在侧边栏中点击 **"发送文件"**。

## 发送文件

### 步骤 1：启用局域网发现
点击标题栏中的 **🌐**（地球）图标。

### 步骤 2：选择文件
1. 转到发送文件 → **发送** 标签
2. 点击 **"选择文件"** 或 **"选择文件夹"**
3. 选中的项目出现在列表中
4. 点击 × 删除项目

### 步骤 3：获取配对码
记下显示的 6 位 **配对码**。

### 步骤 4：分享配对码
告诉接收方您的配对码（口头或通过消息）。

### 步骤 5：开始发送
1. 点击 **"开始发送"**
2. 状态："等待接收方..."
3. 当接收方连接时开始传输

## 接收文件

### 步骤 1：启用局域网发现
点击标题栏中的 **🌐**（地球）图标。

### 步骤 2：连接
1. 转到发送文件 → **接收** 标签
2. 输入发送方的 6 位配对码
3. 点击 **"连接"**

### 步骤 3：接收
- 设备自动连接
- 传输立即开始
- 文件保存到您的下载文件夹

## 传输进度

传输过程中您将看到：
- 带完成百分比的进度条
- 速度指示器（例如"5.2 MB/s"）
- 当前正在传输的文件

## 取消传输

点击 **"取消"** 按钮随时停止传输。

## 故障排除

### 找不到发送方/接收方

1. 确保两台设备都 **启用局域网发现**（🌐 图标激活）
2. 两台设备必须在 **同一本地网络**
3. 检查防火墙是否阻止 Marix
4. 尝试刷新连接

### 传输失败

- 大文件可能需要时间 - 请耐心等待
- 确保网络连接稳定
- 尝试使用新配对码重新发送
- 检查接收方可用磁盘空间

### 传输速度慢

- 使用有线以太网获得更快的传输
- 减少网络拥塞
- 关闭占用带宽的应用程序
- 检查路由器/网络性能

## 安全

- 传输使用 **端到端加密**
- 6 位配对码确保只有预期接收者收到文件
- 没有数据通过外部服务器
- 文件在设备之间直接传输

## 提示

- **大文件夹**：选择文件夹而不是单个文件
- **多个文件**：您可以一次选择多个项目
- **验证接收**：与接收方确认文件已正确到达
- **网络质量**：更好的网络 = 更快的传输

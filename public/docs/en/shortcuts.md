# Keyboard Shortcuts

Complete list of keyboard shortcuts in Marix.

## Notation

| Symbol | Key |
|--------|-----|
| `Ctrl` | Control (Windows/Linux) |
| `⌘` | Command (macOS) |
| `Shift` | Shift |
| `Alt` | Alt/Option |

> **Note:** On macOS, replace `Ctrl` with `⌘` (Command).

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+K` | Open Quick Connect |
| `Ctrl+Shift+A` | Add New Host |
| `Ctrl+Shift+T` | Open Local Terminal |
| `Ctrl+Shift+L` | Toggle LAN Discovery |
| `Ctrl+Shift+O` | Toggle Terminal/SFTP view |

## Terminal Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+C` | Copy (when text selected) / Cancel command |
| `Ctrl+V` | Paste |
| `Ctrl+Shift+C` | Copy (alternative) |
| `Ctrl+Shift+V` | Paste (alternative) |
| `Ctrl+Shift+F` | Find in terminal |
| `Ctrl+L` | Clear screen |
| `Ctrl++` | Zoom in |
| `Ctrl+-` | Zoom out |
| `Ctrl+0` | Reset zoom |

## SFTP Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+Shift+O` | Switch to SFTP |
| `F2` | Rename selected file |
| `Delete` | Delete selected file |
| `Ctrl+C` | Copy file path |
| `Ctrl+R` | Refresh directory |
| `Backspace` | Go to parent directory |

## File Editor Shortcuts

| Shortcut | Action |
|----------|--------|
| `Ctrl+S` | Save file |
| `Ctrl+Z` | Undo |
| `Ctrl+Shift+Z` | Redo |
| `Ctrl+F` | Find |
| `Ctrl+H` | Find and replace |
| `Ctrl+G` | Go to line |

## Window Shortcuts

| Shortcut | Action |
|----------|--------|
| `F11` | Toggle fullscreen |
| `Ctrl+Shift+D` | Toggle dark/light mode |
| `Ctrl+Q` | Quit application |

## Customization

Currently, keyboard shortcuts cannot be customized. This feature may be added in future versions.

## Tips

- Most shortcuts work across all views
- Terminal shortcuts may conflict with shell shortcuts
- Use `Ctrl+Shift+C/V` in terminal to avoid conflicts with `Ctrl+C` (cancel)

# Quick Start Guide

Get connected to your first server in under 5 minutes.

## Add Your First Host

1. Click **add new host** in the server list 
2. Enter server details:
   - **Name**: Any friendly name (e.g., "My VPS")
   - **Protocol**: SSH (default)
   - **Host**: IP address or domain
   - **Port**: 22 (default for SSH)
   - **Username**: Your server username
   - **Authentication**: Password or SSH Key
3. Click **Save**


## Connect to Your Server

1. Click on the host name in server list
2. You're connected!

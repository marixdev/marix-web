# Panduan Cepat

Terhubung ke server pertama Anda dalam waktu kurang dari 5 menit.

## Tambahkan Host Pertama Anda

1. Klik **add new host** di daftar server 
2. Masukkan detail server:
   - **Name**: Nama yang mudah diingat (misalnya, "My VPS")
   - **Protocol**: SSH (default)
   - **Host**: Alamat IP atau domain
   - **Port**: 22 (default untuk SSH)
   - **Username**: Username server Anda
   - **Authentication**: Password atau SSH Key
3. Klik **Save**


## Hubungkan ke Server Anda

1. Klik nama host di daftar server
2. Anda terhubung!

## Tips Cepat

### Pintasan Terminal

| Pintasan | Aksi |
|----------|------|
| `Ctrl+Shift+C` | Copy |
| `Ctrl+Shift+V` | Paste |
| `Ctrl+Shift+F` | Cari |
| `Ctrl+L` | Bersihkan layar |
| `Ctrl+Plus` | Perbesar |
| `Ctrl+Minus` | Perkecil |

### Dukungan Multi-Tab

- **Tab Baru**: Klik + 

### Copy-Paste 

3. **Salin teks**: 
   - Pilih teks dengan mouse → Copy
   - Atau gunakan `Ctrl+C`
4. **Tempel teks**: 
   - Klik kanan → Paste
   - Atau gunakan `Ctrl+V`

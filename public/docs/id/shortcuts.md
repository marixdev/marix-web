# Pintasan Keyboard

Daftar lengkap pintasan keyboard di Marix.

## Notasi

| Simbol | Tombol |
|--------|--------|
| `Ctrl` | Control (Windows/Linux) |
| `⌘` | Command (macOS) |
| `Shift` | Shift |
| `Alt` | Alt/Option |

> **Catatan:** Di macOS, ganti `Ctrl` dengan `⌘` (Command).

## Pintasan Global

| Pintasan | Aksi |
|----------|------|
| `Ctrl+K` | Buka Quick Connect |
| `Ctrl+Shift+A` | Tambah Host Baru |
| `Ctrl+Shift+T` | Buka Terminal Lokal |
| `Ctrl+Shift+L` | Toggle LAN Discovery |
| `Ctrl+Shift+O` | Toggle tampilan Terminal/SFTP |

## Pintasan Terminal

| Pintasan | Aksi |
|----------|------|
| `Ctrl+C` | Copy (saat teks dipilih) / Batalkan perintah |
| `Ctrl+V` | Paste |
| `Ctrl+Shift+C` | Copy (alternatif) |
| `Ctrl+Shift+V` | Paste (alternatif) |
| `Ctrl+Shift+F` | Cari di terminal |
| `Ctrl+L` | Bersihkan layar |
| `Ctrl++` | Perbesar |
| `Ctrl+-` | Perkecil |
| `Ctrl+0` | Reset zoom |

## Pintasan SFTP

| Pintasan | Aksi |
|----------|------|
| `Ctrl+Shift+O` | Beralih ke SFTP |
| `F2` | Ubah nama file terpilih |
| `Delete` | Hapus file terpilih |
| `Ctrl+C` | Salin path file |
| `Ctrl+R` | Refresh direktori |
| `Backspace` | Ke direktori parent |

## Pintasan Editor File

| Pintasan | Aksi |
|----------|------|
| `Ctrl+S` | Simpan file |
| `Ctrl+Z` | Undo |
| `Ctrl+Shift+Z` | Redo |
| `Ctrl+F` | Cari |
| `Ctrl+H` | Cari dan ganti |
| `Ctrl+G` | Pergi ke baris |

## Pintasan Jendela

| Pintasan | Aksi |
|----------|------|
| `F11` | Toggle layar penuh |
| `Ctrl+Shift+D` | Toggle mode gelap/terang |
| `Ctrl+Q` | Keluar aplikasi |

## Kustomisasi

Saat ini, pintasan keyboard tidak dapat dikustomisasi. Fitur ini mungkin akan ditambahkan di versi mendatang.

## Tips

- Sebagian besar pintasan bekerja di semua tampilan
- Pintasan terminal mungkin konflik dengan pintasan shell
- Gunakan `Ctrl+Shift+C/V` di terminal untuk menghindari konflik dengan `Ctrl+C` (batalkan)

## Kemajuan Pemindahan

Semasa pemindahan anda akan melihat:
- Bar kemajuan dengan peratusan penyelesaian
- Penunjuk kelajuan (cth., "5.2 MB/s")
- Fail semasa yang sedang dipindahkan

## Batalkan Pemindahan

Klik butang **"Cancel"** untuk menghentikan pemindahan pada bila-bila masa.

## Penyelesaian Masalah

### Tidak Dapat Mencari Penghantar/Penerima

1. Pastikan kedua-dua peranti mempunyai **LAN Discovery ON** (ikon ğŸŒ aktif)
2. Kedua-dua peranti mesti berada pada **rangkaian tempatan yang sama**
3. Semak firewall tidak menyekat Marix
4. Cuba muat semula sambungan

### Pemindahan Gagal

- Fail besar mungkin mengambil masa - bersabar
- Pastikan sambungan rangkaian stabil
- Cuba hantar semula dengan kod gandingan baru
- Semak ruang cakera yang tersedia pada penerima

### Kelajuan Pemindahan Perlahan

- Gunakan Ethernet berwayar untuk pemindahan lebih pantas
- Kurangkan kesesakan rangkaian
- Tutup aplikasi yang memerlukan lebar jalur tinggi
- Semak prestasi penghala/rangkaian

## Keselamatan

- Pemindahan menggunakan **penyulitan hujung-ke-hujung**
- Kod gandingan 6 digit memastikan hanya penerima yang dimaksudkan menerima fail
- Tiada data melalui pelayan luaran
- Fail dipindahkan terus antara peranti

## Tips

- **Large folders**: Select the folder instead of individual files
- **Multiple files**: You can select multiple items at once
- **Verify receipt**: Confirm with receiver that files arrived correctly
- **Network quality**: Better network = faster transfers

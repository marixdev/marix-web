# Pintasan Papan Kekunci

Senarai lengkap pintasan papan kekunci dalam Marix.

## Notasi

| Simbol | Kekunci |
|--------|-----|
| `Ctrl` | Control (Windows/Linux) |
| `⌘` | Command (macOS) |
| `Shift` | Shift |
| `Alt` | Alt/Option |

> **Nota:** Pada macOS, gantikan `Ctrl` dengan `⌘` (Command).

## Pintasan Global

| Pintasan | Tindakan |
|----------|--------|
| `Ctrl+K` | Buka Quick Connect |
| `Ctrl+Shift+A` | Tambah Hos Baru |
| `Ctrl+Shift+T` | Buka Terminal Tempatan |
| `Ctrl+Shift+L` | Togol LAN Discovery |
| `Ctrl+Shift+O` | Togol paparan Terminal/SFTP |

## Pintasan Terminal

| Pintasan | Tindakan |
|----------|--------|
| `Ctrl+C` | Salin (apabila teks dipilih) / Batalkan arahan |
| `Ctrl+V` | Tampal |
| `Ctrl+Shift+C` | Salin (alternatif) |
| `Ctrl+Shift+V` | Tampal (alternatif) |
| `Ctrl+Shift+F` | Cari dalam terminal |
| `Ctrl+L` | Kosongkan skrin |
| `Ctrl++` | Zum masuk |
| `Ctrl+-` | Zum keluar |
| `Ctrl+0` | Tetap semula zum |

## Pintasan SFTP

| Pintasan | Tindakan |
|----------|--------|
| `Ctrl+Shift+O` | Tukar ke SFTP |
| `F2` | Namakan semula fail yang dipilih |
| `Delete` | Padam fail yang dipilih |
| `Ctrl+C` | Salin laluan fail |
| `Ctrl+R` | Muat semula direktori |
| `Backspace` | Pergi ke direktori induk |

## Pintasan Editor Fail

| Pintasan | Tindakan |
|----------|--------|
| `Ctrl+S` | Simpan fail |
| `Ctrl+Z` | Undo |
| `Ctrl+Shift+Z` | Redo |
| `Ctrl+F` | Cari |
| `Ctrl+H` | Cari dan ganti |
| `Ctrl+G` | Pergi ke baris |

## Pintasan Tetingkap

| Pintasan | Tindakan |
|----------|--------|
| `F11` | Togol skrin penuh |
| `Ctrl+Shift+D` | Togol mod gelap/cerah |
| `Ctrl+Q` | Keluar aplikasi |

## Penyesuaian

Pada masa ini, pintasan papan kekunci tidak boleh disesuaikan. Ciri ini mungkin ditambah dalam versi akan datang.

## Petua

- Kebanyakan pintasan berfungsi merentas semua paparan
- Pintasan terminal mungkin bercanggah dengan pintasan shell
- Gunakan `Ctrl+Shift+C/V` dalam terminal untuk mengelakkan konflik dengan `Ctrl+C` (batalkan)

# Pemasangan

Muat turun dan pasang Marix untuk sistem pengendalian anda.

## Windows

1. Muat turun **[Marix.Setup.{{VERSION}}.exe](https://github.com/marixdev/marix/releases/latest/download/Marix.Setup.{{VERSION}}.exe)**
2. Jalankan pemasang dan ikut wizard
3. Lancarkan Marix dari Menu Start

## macOS

### Apple Silicon (M1/M2/M3/M4)

1. Muat turun **[Marix-{{VERSION}}-arm64-mac.zip](https://github.com/marixdev/marix/releases/latest/download/Marix-{{VERSION}}-arm64-mac.zip)**
2. Ekstrak dan seret `Marix.app` ke Applications
3. Pelancaran pertama: Klik kanan → Open (untuk memintas Gatekeeper)

### Intel Mac

1. Muat turun **[Marix-{{VERSION}}-mac.zip](https://github.com/marixdev/marix/releases/latest/download/Marix-{{VERSION}}-mac.zip)**
2. Ekstrak dan seret `Marix.app` ke Applications
3. Pelancaran pertama: Klik kanan → Open (untuk memintas Gatekeeper)

## Linux

### AppImage (Disyorkan)

Berfungsi pada kebanyakan pengedaran Linux tanpa pemasangan.

1. Muat turun **[Marix-{{VERSION}}.AppImage](https://github.com/marixdev/marix/releases/latest/download/Marix-{{VERSION}}.AppImage)**
2. Jadikan boleh dilaksanakan: `chmod +x Marix-{{VERSION}}.AppImage`
3. Jalankan: `./Marix-{{VERSION}}.AppImage`

### Debian/Ubuntu (.deb)

1. Muat turun **[marix_{{VERSION}}_amd64.deb](https://github.com/marixdev/marix/releases/latest/download/marix_{{VERSION}}_amd64.deb)**
2. Pasang: `sudo dpkg -i marix_{{VERSION}}_amd64.deb`

### Fedora/RHEL (.rpm)

1. Muat turun **[marix-{{VERSION}}.x86_64.rpm](https://github.com/marixdev/marix/releases/latest/download/marix-{{VERSION}}.x86_64.rpm)**
2. Pasang: `sudo dnf install ./marix-{{VERSION}}.x86_64.rpm`

## Sahkan Pemasangan

Lancarkan Marix. Anda sepatutnya melihat tetingkap utama dengan bahagian "Hosts" di bar sisi.

# Panduan Mula Pantas

Sambung ke pelayan pertama anda dalam masa kurang dari 5 minit.

## Tambah Hos Pertama Anda

1. Klik **add new host** dalam senarai pelayan 
2. Masukkan butiran pelayan:
   - **Name**: Sebarang nama mesra (cth., "My VPS")
   - **Protocol**: SSH (lalai)
   - **Host**: Alamat IP atau domain
   - **Port**: 22 (lalai untuk SSH)
   - **Username**: Nama pengguna pelayan anda
   - **Authentication**: Password atau SSH Key
3. Klik **Save**


## Sambung ke Pelayan Anda

1. Klik pada nama hos dalam senarai pelayan
2. Anda disambungkan!

## Petua Pantas

### Pintasan Terminal

| Pintasan | Tindakan |
|----------|--------|
| `Ctrl+Shift+C` | Salin |
| `Ctrl+Shift+V` | Tampal |
| `Ctrl+Shift+F` | Cari |
| `Ctrl+L` | Kosongkan skrin |
| `Ctrl+Plus` | Zum masuk |
| `Ctrl+Minus` | Zum keluar |

### Sokongan Pelbagai Tab

- **Tab Baru**: Klik + 

### Salin-Tampal 

3. **Salin teks**: 
   - Pilih teks dengan tetikus → Salin
   - Atau gunakan `Ctrl+C`
4. **Tampal teks**: 
   - Klik kanan → Paste
   - Atau gunakan `Ctrl+V`

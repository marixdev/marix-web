# Guide de démarrage rapide

Connectez-vous à votre premier serveur en moins de 5 minutes.

## Ajouter votre premier hôte

1. Cliquez sur **ajouter un nouvel hôte** dans la liste des serveurs
2. Entrez les détails du serveur :
   - **Nom** : N'importe quel nom convivial (ex : « Mon VPS »)
   - **Protocole** : SSH (par défaut)
   - **Hôte** : Adresse IP ou domaine
   - **Port** : 22 (par défaut pour SSH)
   - **Nom d'utilisateur** : Votre nom d'utilisateur serveur
   - **Authentification** : Mot de passe ou clé SSH
3. Cliquez sur **Enregistrer**


## Se connecter à votre serveur

1. Cliquez sur le nom de l'hôte dans la liste des serveurs
2. Vous êtes connecté !

## Astuces rapides

### Raccourcis du terminal

| Raccourci | Action |
|-----------|--------|
| `Ctrl+Shift+C` | Copier |
| `Ctrl+Shift+V` | Coller |
| `Ctrl+Shift+F` | Rechercher |
| `Ctrl+L` | Effacer l'écran |
| `Ctrl+Plus` | Zoom avant |
| `Ctrl+Minus` | Zoom arrière |

### Support multi-onglets

- **Nouvel onglet** : Cliquez sur +

### Copier-Coller

3. **Copier du texte** :
   - Sélectionnez le texte avec la souris → Copier
   - Ou utilisez `Ctrl+C`
4. **Coller du texte** :
   - Clic droit → Coller
   - Ou utilisez `Ctrl+V`

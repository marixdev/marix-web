# Installation

Laden Sie Marix herunter und installieren Sie es für Ihr Betriebssystem.

## Windows

1. Laden Sie **[Marix.Setup.{{VERSION}}.exe](https://github.com/marixdev/marix/releases/latest/download/Marix.Setup.{{VERSION}}.exe)** herunter
2. Führen Sie das Installationsprogramm aus und folgen Sie dem Assistenten
3. Starten Sie Marix aus dem Startmenü

## macOS

### Apple Silicon (M1/M2/M3/M4)

1. Laden Sie **[Marix-{{VERSION}}-arm64-mac.zip](https://github.com/marixdev/marix/releases/latest/download/Marix-{{VERSION}}-arm64-mac.zip)** herunter
2. Entpacken Sie die Datei und ziehen Sie `Marix.app` in Programme
3. Erster Start: Rechtsklick → Öffnen (um Gatekeeper zu umgehen)

### Intel Mac

1. Laden Sie **[Marix-{{VERSION}}-mac.zip](https://github.com/marixdev/marix/releases/latest/download/Marix-{{VERSION}}-mac.zip)** herunter
2. Entpacken Sie die Datei und ziehen Sie `Marix.app` in Programme
3. Erster Start: Rechtsklick → Öffnen (um Gatekeeper zu umgehen)

## Linux

### AppImage (Empfohlen)

Funktioniert auf den meisten Linux-Distributionen ohne Installation.

1. Laden Sie **[Marix-{{VERSION}}.AppImage](https://github.com/marixdev/marix/releases/latest/download/Marix-{{VERSION}}.AppImage)** herunter
2. Ausführbar machen: `chmod +x Marix-{{VERSION}}.AppImage`
3. Ausführen: `./Marix-{{VERSION}}.AppImage`

### Debian/Ubuntu (.deb)

1. Laden Sie **[marix_{{VERSION}}_amd64.deb](https://github.com/marixdev/marix/releases/latest/download/marix_{{VERSION}}_amd64.deb)** herunter
2. Installieren: `sudo dpkg -i marix_{{VERSION}}_amd64.deb`

### Fedora/RHEL (.rpm)

1. Laden Sie **[marix-{{VERSION}}.x86_64.rpm](https://github.com/marixdev/marix/releases/latest/download/marix-{{VERSION}}.x86_64.rpm)** herunter
2. Installieren: `sudo dnf install ./marix-{{VERSION}}.x86_64.rpm`

## Installation überprüfen

Starten Sie Marix. Sie sollten das Hauptfenster mit dem Abschnitt „Hosts" in der Seitenleiste sehen.

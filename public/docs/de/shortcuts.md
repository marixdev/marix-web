# Tastenkürzel

Vollständige Liste der Tastenkürzel in Marix.

## Notation

| Symbol | Taste |
|--------|-------|
| `Strg` | Control (Windows/Linux) |
| `⌘` | Command (macOS) |
| `Shift` | Umschalttaste |
| `Alt` | Alt/Option |

> **Hinweis:** Unter macOS ersetzen Sie `Strg` durch `⌘` (Command).

## Globale Tastenkürzel

| Tastenkürzel | Aktion |
|--------------|--------|
| `Strg+K` | Schnellverbindung öffnen |
| `Strg+Shift+A` | Neuen Host hinzufügen |
| `Strg+Shift+T` | Lokales Terminal öffnen |
| `Strg+Shift+L` | LAN-Erkennung umschalten |
| `Strg+Shift+O` | Terminal/SFTP-Ansicht umschalten |

## Terminal-Tastenkürzel

| Tastenkürzel | Aktion |
|--------------|--------|
| `Strg+C` | Kopieren (bei Textauswahl) / Befehl abbrechen |
| `Strg+V` | Einfügen |
| `Strg+Shift+C` | Kopieren (Alternative) |
| `Strg+Shift+V` | Einfügen (Alternative) |
| `Strg+Shift+F` | Im Terminal suchen |
| `Strg+L` | Bildschirm löschen |
| `Strg++` | Vergrößern |
| `Strg+-` | Verkleinern |
| `Strg+0` | Zoom zurücksetzen |

## SFTP-Tastenkürzel

| Tastenkürzel | Aktion |
|--------------|--------|
| `Strg+Shift+O` | Zu SFTP wechseln |
| `F2` | Ausgewählte Datei umbenennen |
| `Entf` | Ausgewählte Datei löschen |
| `Strg+C` | Dateipfad kopieren |
| `Strg+R` | Verzeichnis aktualisieren |
| `Rücktaste` | Zum übergeordneten Verzeichnis |

## Datei-Editor-Tastenkürzel

| Tastenkürzel | Aktion |
|--------------|--------|
| `Strg+S` | Datei speichern |
| `Strg+Z` | Rückgängig |
| `Strg+Shift+Z` | Wiederholen |
| `Strg+F` | Suchen |
| `Strg+H` | Suchen und ersetzen |
| `Strg+G` | Gehe zu Zeile |

## Fenster-Tastenkürzel

| Tastenkürzel | Aktion |
|--------------|--------|
| `F11` | Vollbildmodus umschalten |
| `Strg+Shift+D` | Hell/Dunkel-Modus umschalten |
| `Strg+Q` | Anwendung beenden |

## Anpassung

Derzeit können Tastenkürzel nicht angepasst werden. Diese Funktion könnte in zukünftigen Versionen hinzugefügt werden.

## Tipps

- Die meisten Tastenkürzel funktionieren in allen Ansichten
- Terminal-Tastenkürzel können mit Shell-Tastenkürzel kollidieren
- Verwenden Sie `Strg+Shift+C/V` im Terminal, um Konflikte mit `Strg+C` (Abbrechen) zu vermeiden

# Schnellstart-Anleitung

Verbinden Sie sich in weniger als 5 Minuten mit Ihrem ersten Server.

## Ersten Host hinzufügen

1. Klicken Sie auf **Neuen Host hinzufügen** in der Serverliste 
2. Geben Sie die Serverdetails ein:
   - **Name**: Beliebiger Anzeigename (z.B. „Mein VPS")
   - **Protokoll**: SSH (Standard)
   - **Host**: IP-Adresse oder Domain
   - **Port**: 22 (Standard für SSH)
   - **Benutzername**: Ihr Server-Benutzername
   - **Authentifizierung**: Passwort oder SSH-Schlüssel
3. Klicken Sie auf **Speichern**


## Mit Ihrem Server verbinden

1. Klicken Sie auf den Hostnamen in der Serverliste
2. Sie sind verbunden!

## Schnelle Tipps

### Terminal-Tastenkürzel

| Tastenkürzel | Aktion |
|--------------|--------|
| `Strg+Shift+C` | Kopieren |
| `Strg+Shift+V` | Einfügen |
| `Strg+Shift+F` | Suchen |
| `Strg+L` | Bildschirm löschen |
| `Strg+Plus` | Vergrößern |
| `Strg+Minus` | Verkleinern |

### Multi-Tab-Unterstützung

- **Neuer Tab**: Klicken Sie auf + 

### Kopieren-Einfügen 

3. **Text kopieren**: 
   - Text mit Maus auswählen → Kopieren
   - Oder `Strg+C` verwenden
4. **Text einfügen**: 
   - Rechtsklick → Einfügen
   - Oder `Strg+V` verwenden
